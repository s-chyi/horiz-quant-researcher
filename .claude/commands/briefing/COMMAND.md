# /briefing — 晨报/复盘命令

## 功能

盘前: 生成隔夜要闻、全球市场、资金动向、策略建议的综合简报。
盘后: 复盘当日行情，分析资金行为，展望次日。

## 执行流程

1. **判断时段**: 根据当前时间或用户指定判断盘前/盘后
   - 盘前 (9:30前): 侧重隔夜全球市场、资金预期、策略建议
   - 盘后 (15:00后): 侧重当日复盘、资金回顾、次日展望
   - 盘中: 提示当前盘中，给出实时快照

2. **数据采集** (并行):
   - market-data: 全球指数行情
   - fund-flow: 资金流向数据
   - news-sentiment: 最新新闻和舆情
   - Web搜索: 隔夜重要事件

3. **技能调用**:
   - 主技能: morning-briefing
   - 辅助: macro-transmission (宏观面判断)
   - 辅助: catalyst-scanner (当日重要事件)

4. **输出格式**: 按 morning-briefing 技能的标准5步格式输出

## 自动增强

- 如果用户之前讨论过特定个股/行业，自动在简报中加入相关标的的更新
- 如果检测到重大异动，自动升级为详细分析
