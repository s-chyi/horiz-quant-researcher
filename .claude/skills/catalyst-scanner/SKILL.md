# 催化剂扫描

## 适用场景

识别和评估可能改变投资标的价格或行业走势的催化事件。

## 执行步骤

### 步骤 1: 催化剂分类扫描

```
催化剂全景扫描:

├── 📊 公司层面
│   ├── 业绩相关: 财报发布 / 业绩预告 / 盈利修正
│   ├── 经营事件: 新产品发布 / 大客户获取 / 产能投放
│   ├── 资本运作: 增发 / 回购 / 分红 / 并购重组
│   ├── 管理层变动: 高管更替 / 股权激励
│   └── 股东行为: 大股东增减持 / 股权质押
│
├── 📈 行业层面
│   ├── 供需变化: 产能投放 / 需求拐点 / 库存周期
│   ├── 价格变化: 产品涨价 / 原材料价格
│   ├── 技术变革: 新技术突破 / 技术路线切换
│   └── 竞争格局: 龙头策略变化 / 新进入者
│
├── 🏛️ 政策层面
│   ├── 产业政策: 扶持 / 限制 / 准入
│   ├── 财税政策: 税收优惠 / 补贴 / 退坡
│   ├── 监管政策: 行业监管 / 反垄断 / 环保
│   └── 贸易政策: 关税 / 制裁 / 出口管制
│
└── 🌍 宏观层面
    ├── 货币政策: 降息 / 降准 / 流动性
    ├── 经济数据: GDP / PMI / CPI / 社融
    ├── 地缘政治: 国际关系 / 区域冲突
    └── 汇率变动: 人民币升值/贬值
```

### 步骤 2: 催化剂评估矩阵

```
催化剂评估表:

序号 | 催化剂描述      | 类型  | 方向 | 概率  | 影响  | 时间窗口  | 关注度
────┼────────────────┼──────┼─────┼──────┼──────┼─────────┼──────
  1  | [具体事件]      | 公司  | 正/负| [X]% | 大中小| [时间]   | ★★★★★
  2  | [具体事件]      | 行业  | 正/负| [X]% | 大中小| [时间]   | ★★★★☆
  3  | [具体事件]      | 政策  | 正/负| [X]% | 大中小| [时间]   | ★★★☆☆
  ...
```

### 步骤 3: 影响路径推演

对于高关注度催化剂，推演完整影响路径:

```
催化剂: [事件描述]

如果触发:
  一阶效应: [直接影响] (时滞: [X]天/周/月)
    → 二阶效应: [传导影响] (时滞: [X])
      → 三阶效应: [最终对标的影响] (时滞: [X])
        → 预计股价影响: [+/-X%] (基于历史类比)

如果未触发:
  → 替代情景: [...]
  → 对投资论点的影响: [是否改变核心逻辑]

历史类比:
  [类似催化剂] 在 [时间] 触发后:
    → 股价反应: [X]% / [Y]个交易日
    → 持续性: [短暂冲击 / 趋势性改变]
```

### 步骤 5: ASCII 催化剂地图

```
示例:

  催化剂时间线 & 影响矩阵

  影响
  大 │     ●财报(4/25)          ●新品发布(6月)
    │          ○降息预期
  中 │  ●行业数据              ●产能投放
    │              ○关税谈判
  小 │      ○高管增持     ○分红

  正面催化剂: 绿色  负面催化剂: 红色  不确定: 黄色
```

### 步骤 6: Variant Perception (VP) 因子提取

参考 SOIC 方法论，对每个催化剂提取「变量认知差」:

```
VP 因子评分卡:

因子类别          | 具体因子            | 市场共识    | 我们的判断  | VP 差距 | 概率×影响
─────────────────┼────────────────────┼───────────┼───────────┼────────┼─────────
新产品放量        | [具体产品/市场]     | [共识看法]  | [我们看法]  | [高/中/低]| [X]
地理扩张          | [具体区域/计划]     | [共识看法]  | [我们看法]  | [高/中/低]| [X]
终端行业增长      | [下游行业趋势]      | [共识看法]  | [我们看法]  | [高/中/低]| [X]
客户钱包份额提升  | [大客户深耕]        | [共识看法]  | [我们看法]  | [高/中/低]| [X]
市场份额抢夺      | [竞争优势变化]      | [共识看法]  | [我们看法]  | [高/中/低]| [X]

VP 核心结论: 市场对 [X因子] 低估程度最大，构成最主要的正向催化剂。
```

**排序规则**: VP 差距 × 概率 × 影响 → 取 Top 3 做深度推演。

### 步骤 7: 前瞻性触发器清单

从路演纪要和研报中提取可验证的、有时间节点的触发器:

```
可追踪触发器:

优先级 | 触发器描述             | 验证时间    | 数据源     | 如果触发→ | 如果落空→
──────┼──────────────────────┼───────────┼──────────┼─────────┼──────────
P0    | [季度业绩超预期>10%]   | [Q1财报]   | 财报/路演  | 买入信号   | 重新评估
P0    | [新产品订单确认]       | [具体月份]  | 公告/路演  | 加仓催化   | VP 失效
P1    | [行业政策落地]         | [两会后]   | 政府网站   | 板块机会   | 延后关注
```

## 数据源

1. **horiz-datasource-api** → 路演纪要中的管理层展望、管理层 Q&A 中的前瞻性指引
2. **AKShare API (localhost:8901)** → 个股资金流向、股东变化等催化信号
3. **Web 搜索** → 最新政策、公告、行业动态
4. **A 股重要日历** → `references/a-share-calendar.md`
5. **讯兔热点话题** → 当前市场关注的催化主题
