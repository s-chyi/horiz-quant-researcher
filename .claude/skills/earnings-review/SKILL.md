# 业绩点评

## 适用场景

公司发布财报（季报/半年报/年报/业绩预告/快报）后，对业绩进行深度解读。

## 核心原则

**不仅告诉用户数字是什么，更要告诉用户数字意味着什么。**

## 执行步骤

### 步骤 1: 核心数据提取

```
关键指标速览:
┌────────────────┬──────────┬──────────┬──────────┐
│ 指标            │ 本期      │ 上期      │ YoY/QoQ  │
├────────────────┼──────────┼──────────┼──────────┤
│ 营业收入        │          │          │          │
│ 归母净利润      │          │          │          │
│ 扣非净利润      │          │          │          │
│ 经营性现金流    │          │          │          │
│ 毛利率          │          │          │ +/- pp   │
│ 净利率          │          │          │ +/- pp   │
│ ROE (年化)      │          │          │          │
└────────────────┴──────────┴──────────┴──────────┘
```

### 步骤 2: 超预期分析

```
业绩 vs 预期对比:
┌────────────────┬──────────┬──────────┬──────────┐
│ 指标            │ 实际值    │ 市场预期  │ 偏差     │
├────────────────┼──────────┼──────────┼──────────┤
│ 营收            │          │          │ +/-[X]%  │
│ 净利润          │          │          │ +/-[X]%  │
│ 毛利率          │          │          │ +/- pp   │
└────────────────┴──────────┴──────────┴──────────┘

超预期/低于预期的原因拆解:
  → 量的因素 (销量/出货量变化)
  → 价的因素 (ASP/单价变化)
  → 成本的因素 (原材料/人工/折旧)
  → 费用的因素 (销售/管理/研发投入)
  → 非经常性因素 (政府补贴/资产处置/投资收益)
```

### 步骤 3: 业绩质量评估

```
质量检查清单:
├── 1. 营收质量
│   ├── 营收增长是否有合同/订单支撑？
│   ├── 应收账款增速 vs 营收增速 → 超过营收增速则⚠️
│   └── 前五大客户集中度变化
│
├── 2. 利润质量
│   ├── 扣非净利润 vs 归母净利润 → 差异过大则⚠️
│   ├── 经营现金流 / 净利润 → <0.7 则含金量不足
│   └── 政府补贴/投资收益占比
│
├── 3. 资产负债质量
│   ├── 存货变化 → 大幅增长可能是滞销信号
│   ├── 商誉减值风险
│   ├── 有息负债变化 → 是否在加杠杆
│   └── 合同负债变化 → 预收增加是积极信号
│
└── 4. 现金流质量
    ├── 经营活动现金流趋势
    ├── 自由现金流 = 经营现金流 - 资本开支
    └── 资本开支方向 → 维持性 vs 扩张性
```

### 步骤 4: 分业务/分区域拆解

```
业务线拆解:
  业务A: 营收[X]亿 (占比[Y]%, YoY [Z]%) → [增减原因]
  业务B: 营收[X]亿 (占比[Y]%, YoY [Z]%) → [增减原因]
  ...

利润贡献拆解:
  业务A: 毛利率[X]%, 贡献毛利[Y]亿 → [变动原因]
  业务B: 毛利率[X]%, 贡献毛利[Y]亿 → [变动原因]
  ...

区域拆解 (如适用):
  国内: 占比[X]%, YoY [Y]%
  海外: 占比[X]%, YoY [Y]% → [重点市场表现]
```

### 步骤 5: 前瞻性判断

```
管理层指引:
  → 管理层对下季度/全年的展望
  → 历史兑现率（管理层过去的指引准确度如何？）

关键前瞻指标:
  ├── 合同负债/预收账款趋势 → 领先指标
  ├── 新签订单/在手订单 → 收入可见度
  ├── 资本开支计划 → 未来产能
  ├── 研发投入方向 → 未来增长点
  └── 行业景气度指标 → 外部环境

下一季度核心看点:
  1. [需要跟踪的关键变量]
  2. [需要跟踪的关键变量]
  3. [需要跟踪的关键变量]
```

### 步骤 6: 对估值的影响

```
业绩修正影响:
  调整前 EPS 预期: [X] 元
  调整后 EPS 预期: [X] 元 (变动 [Y]%)

  当前 PE [X]x → 隐含市场对未来增速预期为 [Y]%
  业绩指向的合理 PE 范围: [X]-[Y]x
  对应价格区间: [X]-[Y] 元
```

### 步骤 7: 生成业绩速评 ASCII 图

```
示例:
  ┌──────────── 2025Q3 业绩速评 ────────────┐
  │                                          │
  │  营收: 421亿 (+28.3% YoY) ✅ 超预期5%    │
  │  净利: 105亿 (+15.2% YoY) ⚠️ 略低预期2%  │
  │                                          │
  │  ┌──────── 业绩质量评估 ────────┐         │
  │  │ 营收质量: ★★★★☆ (应收可控)  │         │
  │  │ 利润质量: ★★★☆☆ (补贴占比↑) │         │
  │  │ 现金流:   ★★★★★ (经营CF强劲) │         │
  │  └────────────────────────────┘         │
  │                                          │
  │  核心矛盾: 量增价减，成本端抢跑利润端     │
  │  关键跟踪: 碳酸锂价格 / 海外出货占比      │
  └──────────────────────────────────────────┘
```

## 数据源

1. horiz-datasource-api → 查询相关路演纪要（公司管理层交流）
2. Web 搜索 → 最新财报数据和券商速评
3. 腾讯/新浪 API → 实时行情（看市场反应）
