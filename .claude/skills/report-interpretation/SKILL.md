# 研报解读

## 适用场景

用户提供一份券商研报（或研报要点），需要快速提取核心逻辑、验证假设合理性、识别分析师偏见和盲点。

## 核心原则

**研报是卖方观点，天然存在乐观偏差。解读的核心不是复述，而是验证逻辑、质疑假设、找到盲点。**

## 执行步骤

### 步骤 1: 研报要素提取

```
研报基本信息:

标题: [研报标题]
机构: [券商名称]
分析师: [姓名] ([从业年限/过往准确率，如可查])
发布日期: [日期]
评级: [买入/增持/中性/减持/卖出] (前次评级: [X])
目标价: [X]元 (前次目标价: [X]元, 当前股价: [X]元)
目标价隐含涨幅: [X]%

核心观点 (逐条提取):
1. [观点1] ← 依据: [...]
2. [观点2] ← 依据: [...]
3. [观点3] ← 依据: [...]

盈利预测:
        | FY[X]A  | FY[X]E  | FY[X]E  | FY[X]E
营收(亿) |         |         |         |
YoY%    |         |         |         |
净利(亿) |         |         |         |
YoY%    |         |         |         |
EPS(元)  |         |         |         |
PE(x)   |         |         |         |
```

### 步骤 2: 逻辑链验证

```
逻辑链拆解与验证:

核心投资逻辑:
[用一句话概括分析师的核心逻辑]

逻辑链:
假设A: [前提假设] → 是否成立? [✅合理 / ⚠️存疑 / ❌不合理]
  │    验证: [支持/反驳这个假设的证据]
  ↓
推论B: [从A推导出的结论] → 推导是否严谨? [✅严谨 / ⚠️跳跃 / ❌断裂]
  │    问题: [推导过程中可能的逻辑漏洞]
  ↓
推论C: [进一步推导] → 推导是否严谨? [✅/⚠️/❌]
  │    问题: [...]
  ↓
结论: [最终结论/目标价/评级]

逻辑链完整性评分: [X]/10
  ├── 前提假设合理性: [X]/10
  ├── 推导过程严谨性: [X]/10
  ├── 数据支撑充分性: [X]/10
  └── 结论与论证一致性: [X]/10
```

### 步骤 3: 偏见与盲点识别

```
偏见扫描:

├── 利益冲突检查:
│   ├── 该券商是否为标的主承销商/保荐人? [是/否/未知]
│   ├── 该分析师历史评级分布: [买入X%/增持X%/中性X%/减持X%]
│   ├── 该分析师对该股历史准确率: [如可查]
│   └── 利益冲突风险: [高/中/低]
│
├── 常见偏见检测:
│   ├── 锚定偏见: 是否过度依赖历史数据外推? [是/否]
│   ├── 确认偏见: 是否只引用支持结论的证据? [是/否]
│   ├── 乐观偏见: 盈利预测是否显著高于市场一致预期? [是/否]
│   ├── 从众偏见: 观点是否与市场共识过度一致? [是/否]
│   └── 线性外推: 是否简单将近期趋势延续? [是/否]
│
├── 盲点识别:
│   ├── 研报未提及但重要的风险:
│   │   1. [盲点1] — 为什么重要: [...]
│   │   2. [盲点2] — 为什么重要: [...]
│   │
│   ├── 研报未讨论的替代情景:
│   │   1. [替代情景] — 如果发生: [影响]
│   │
│   └── 数据缺失:
│       1. [应该有但没有引用的数据]
│       2. [数据的时效性问题]
│
└── 偏见综合评级: [低偏见/中度偏见/高偏见/严重偏见]
```

### 步骤 4: 关键假设敏感性

```
关键假设敏感性测试:

分析师的目标价高度依赖以下假设:

假设 #1: [假设内容]
  ├── 分析师假设值: [X]
  ├── 若乐观 (+20%): 目标价变为 [X]元 (影响 [+/-X]%)
  ├── 若悲观 (-20%): 目标价变为 [X]元 (影响 [+/-X]%)
  └── 敏感度: [高/中/低] (目标价对此假设的弹性)

假设 #2: [假设内容]
  ├── 分析师假设值: [X]
  ├── 若乐观: ...
  ├── 若悲观: ...
  └── 敏感度: [高/中/低]

最敏感假设: [X] — 这是判断研报靠不靠谱的关键变量
```

### 步骤 5: 独立评价

```
独立评价:

与市场一致预期对比:
├── 分析师营收预测 vs 一致预期: [高于/低于/持平] [X]%
├── 分析师利润预测 vs 一致预期: [高于/低于/持平] [X]%
├── 分析师目标价 vs 一致预期: [高于/低于/持平] [X]%
└── 偏离度判断: [在合理范围内 / 偏乐观 / 偏悲观 / 激进]

独立结论:
├── 研报质量评分: [X]/10
│   ├── 逻辑完整性: [X]/10
│   ├── 数据支撑度: [X]/10
│   ├── 独立性: [X]/10
│   └── 实用价值: [X]/10
│
├── 研报结论可信度: [高/中/低]
│   └── 理由: [为什么我认为可信/不可信]
│
├── 我的独立判断:
│   ├── 同意的部分: [...]
│   ├── 不同意的部分: [...]
│   └── 分析师可能忽略的: [...]
│
└── 建议:
    ├── 如果认同研报逻辑: [操作建议]
    ├── 如果持保留态度: [替代方案]
    └── 关键验证时间点: [什么时候可以验证研报假设]
```
