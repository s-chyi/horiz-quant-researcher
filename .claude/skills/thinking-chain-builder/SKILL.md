# 思维链构建器

## 功能

为每类投研问题构建专属的思维链模板，确保推理过程既有深度又有结构。这是超越 PaiPai 的核心引擎。

## 思维链模式库

### 模式 A: 演绎链 (Deductive Chain)

适用于: 个股分析、估值分析
```
大前提 (行业/宏观逻辑)
  → 小前提 (公司具体情况)
    → 推论 (当前定位判断)
      → 验证 (数据交叉检验)
        → 结论 (含置信度)
```

示例:
```
大前提: 全球锂电产业正从供不应求转向产能过剩（产能增速 40% vs 需求增速 25%）
  → 小前提: 宁德时代市占率 37% 但产能利用率从 95% 降至 70%
    → 推论: 量增价减是大概率，关键看成本控制能力
      → 验证: Q3 毛利率 22.3% vs Q2 21.8%，环比改善 → 成本端确有优势
        → 结论: 短期业绩承压但竞争优势未削弱 [置信度: 中高]
```

### 模式 B: 辩证链 (Dialectic Chain)

适用于: 多空辩证、争议性问题
```
正题 (Thesis): 多头核心论点
  → 反题 (Antithesis): 空头核心论点
    → 合题 (Synthesis): 矛盾的深层原因
      → 关键变量: 决定胜负的核心因子
        → 概率评估: 各情景概率分布
```

### 模式 C: 传导链 (Transmission Chain)

适用于: 宏观传导、政策分析
```
触发事件 (Trigger)
  → 一阶效应 (Direct Impact): 直接受影响的变量
    → 二阶效应 (Indirect Impact): 传导至相关领域
      → 三阶效应 (Downstream): 最终对标的的影响
        → 时滞估计: 各阶传导需要多长时间
```

### 模式 D: 归纳链 (Inductive Chain)

适用于: 行业比较、板块分析
```
数据收集 (多个观测样本)
  → 模式识别 (共同特征提取)
    → 异常标记 (偏离模式的个案)
      → 规律提炼 (一般性结论)
        → 预测推广 (应用于新场景)
```

### 模式 E: 拆解链 (Decomposition Chain)

适用于: 业绩分析、估值拆解
```
总量指标 (如: 营收)
  → 一级拆解 (业务线/产品线)
    → 二级拆解 (量×价 / 区域)
      → 驱动因子识别 (增长来源)
        → 可持续性判断 (一次性 vs 持续性)
```

### 模式 F: 情景链 (Scenario Chain)

适用于: 风险分析、前瞻预测
```
基准情景 (Base Case): P=50%
  → 乐观情景 (Bull Case): P=25%
    → 悲观情景 (Bear Case): P=20%
      → 极端情景 (Tail Risk): P=5%
        → 期望值计算 + 最大损失评估
```

## 链式组合规则

复杂问题通常需要组合多条思维链:

```
个股深度分析 = 拆解链(业绩) + 演绎链(逻辑) + 辩证链(多空) + 情景链(估值)
行业链分析 = 传导链(上下游) + 归纳链(公司对比) + 拆解链(利润分配)
宏观传导 = 传导链(政策→行业) + 演绎链(行业→个股) + 情景链(不确定性)
```

## ASCII 结构图要求

每个分析必须包含至少一个 ASCII 结构图，展示思维链的**关键节点和关系**:

```
可选图形:
├── 因果环路图 (Causal Loop Diagram)
├── 2×2 矩阵 (Matrix)
├── 光谱轴 (Spectrum)
├── 层级树 (Hierarchy Tree)
├── 流程图 (Flow Chart)
├── 时间线 (Timeline)
└── 网络拓扑 (Network Topology)
```

图形选择原则: **哪种最能展示分析结构就用哪种**，不固定格式。
