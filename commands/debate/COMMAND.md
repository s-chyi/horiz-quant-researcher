# /debate — 多空辩论命令

## 功能

对投资标的或市场观点进行结构化多空辩论。强制展示双方最强论点，避免单边偏见。

## 执行流程

1. **明确辩题**: 从用户输入中提取辩论主题
   - "[股票]值不值得买" → 辩题: 该股票是否值得投资
   - "[行业]是否见顶" → 辩题: 该行业是否已经见顶
   - "大盘会不会跌" → 辩题: 市场短期是否面临调整风险

2. **数据采集** (并行):
   - datasource-query: 相关路演/研报中的多空观点
   - news-sentiment: 市场情绪和关键新闻
   - market-data: 行情和估值背景
   - Web搜索: 各方观点收集

3. **技能调用**:
   - 主技能: bull-bear-debate
   - 辅助: risk-matrix (为空方提供风险论据)
   - 辅助: catalyst-scanner (为多方提供催化剂论据)
   - 辅助: valuation-framework (估值层面的多空依据)

4. **质量要求**:
   - 多空双方至少各3个核心论点
   - 每个论点必须有数据支撑
   - 不允许稻草人谬误（反方论点必须是强有力的）
   - 必须给出概率判断和关键变量
