# /macro — 宏观分析命令

## 功能

分析宏观经济环境、政策变化，推演对行业和个股的传导路径。

## 执行流程

1. **宏观主题识别**: 确定用户关注的宏观变量
   - 货币政策类: 降息/降准/流动性
   - 财政政策类: 专项债/基建/减税
   - 经济数据类: GDP/PMI/CPI/社融
   - 外部环境类: 美联储/贸易/地缘

2. **数据采集** (并行):
   - Web搜索: 最新宏观数据和政策
   - news-sentiment: 政策相关新闻
   - market-data: 利率/汇率/商品数据

3. **技能调用**:
   - 主技能: macro-transmission
   - 辅助: morning-briefing (全球市场背景)
   - 辅助: catalyst-scanner (政策相关催化剂)

4. **输出增强**:
   - 自动推演传导链到具体行业和板块
   - 给出历史类比参考
   - 标注各阶传导的预期时滞
