# /risk-check — 风险检查命令

## 功能

快速扫描指定标的或投资组合的风险因素，构建概率-影响矩阵。

## 执行流程

1. **风险标的确认**: 识别用户关注的标的
   - 单个股票: 全面风险扫描
   - 多只股票: 组合风险评估
   - 行业/板块: 系统性风险评估

2. **数据采集** (并行):
   - datasource-query: 路演/研报中提及的风险因素
   - news-sentiment: 负面新闻和舆情
   - market-data: 技术面异常信号
   - fund-flow: 资金异常流动
   - Web搜索: 财务风险指标、诉讼信息

3. **技能调用**:
   - 主技能: risk-matrix
   - 辅助: news-sentiment (舆情风险)
   - 辅助: fund-flow (资金面风险信号)

4. **输出增强**:
   - 自动检查近期是否有异常信号（如高管减持、商誉减值预警等）
   - 标注每个风险的触发概率和可能的影响幅度
   - 给出明确的风险应对建议和止损建议
