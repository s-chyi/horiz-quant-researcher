# /valuation — 估值分析命令

## 功能

对标的进行多方法估值分析，给出合理估值区间。

## 执行流程

1. **标的确认**: 识别待估值公司
2. **公司分类**: 确定公司类型（成熟盈利/高速成长/周期/金融等）
3. **方法选择**: 根据公司类型自动选择适用的估值方法

4. **数据采集** (并行):
   - market-data: 当前估值指标
   - datasource-query: 路演/研报中的盈利预测
   - Web搜索: 可比公司数据、历史估值、一致预期

5. **技能调用**:
   - 主技能: valuation-framework
   - 辅助: comparative-analysis (可比公司法需要)
   - 辅助: earnings-review (盈利预测基础)

6. **质量要求**:
   - 至少使用3种估值方法交叉验证
   - DCF必须包含敏感性分析
   - 明确标注每个假设的依据
   - 给出估值区间而非单一数字
