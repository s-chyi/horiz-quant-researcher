# A股重要日历参考手册

## 一、交易时间

```
A股交易时间:
├── 集合竞价: 09:15 - 09:25 (09:15-09:20可撤单，09:20-09:25不可撤单)
├── 连续竞价: 09:30 - 11:30, 13:00 - 15:00
├── 盘后固定价格交易: 15:05 - 15:30 (收盘价成交)
└── 大宗交易: 15:05 - 15:30 (申报)

港股通交易时间:
├── 开市前: 09:00 - 09:30
├── 持续交易: 09:30 - 12:00, 13:00 - 16:00
└── 注意: 港股通仅在A+H两地同时开市时可交易

北向资金:
├── 交易时间与A股同步
├── 每日额度: 无限制 (2023年取消)
└── 实时数据: 9:30开始，10分钟延迟，盘后可查全天明细
```

## 二、年度重要节点

### 财报披露日历
```
年报 (上一年度):
├── 披露窗口: 1月1日 - 4月30日
├── 业绩预告: 1月31日前 (满足条件的公司)
├── 业绩快报: 通常2-3月陆续发布
├── 正式年报: 4月30日截止
├── 关键月份: 3-4月是年报密集期
└── 注意: 4月底不披露 → 可能有问题 ⚠️

一季报:
├── 披露窗口: 4月1日 - 4月30日
├── 业绩预告: 4月15日前 (满足条件的公司)
└── 注意: 一季报与年报同月，信息密集

半年报 (中报):
├── 披露窗口: 7月1日 - 8月31日
├── 业绩预告: 7月15日前 (满足条件的公司)
└── 关键月份: 7-8月是中报期

三季报:
├── 披露窗口: 10月1日 - 10月31日
├── 业绩预告: 10月15日前 (满足条件的公司)
└── 关键月份: 10月

业绩预告强制披露条件:
├── 净利润为负
├── 净利润同比变化≥50%
├── 扭亏为盈
└── 由盈转亏

财报行情规律:
├── 1-2月: 年报预告炒作期 (高增长预告受追捧)
├── 3-4月: 年报+一季报验证期 (超预期/不及预期分化)
├── 7-8月: 中报行情 (关注业绩反转标的)
├── 10月: 三季报行情 (全年业绩基本确定)
└── 11-12月: 来年业绩展望/估值切换
```

### 政策日历
```
全国两会:
├── 时间: 每年3月初 (通常3月3日-3月15日左右)
├── 内容: 政府工作报告、年度经济目标、重点政策方向
├── 市场影响:
│   ├── 两会前: 政策预期升温，"两会行情"
│   ├── 两会中: 关注报告措辞和新提法
│   └── 两会后: 政策细则陆续落地
└── 重点关注: GDP目标、赤字率、专项债额度、产业政策

政治局会议 (经济专题):
├── 4月: 一季度经济形势分析 → 全年政策微调窗口
├── 7月: 半年经济形势分析 → 下半年政策定调
├── 10月: (有时合并或简短)
├── 12月: 中央经济工作会议前的定调
└── 市场影响: 是政策转向最重要的信号源

中央经济工作会议:
├── 时间: 每年12月中旬
├── 内容: 次年经济工作总体部署
├── 重要性: ⭐⭐⭐⭐⭐ (年度最重要的经济政策会议)
└── 关注: 对明年经济定性、财政货币政策基调

其他重要会议:
├── 国务院常务会议: 每周三 (高频政策信号)
├── 央行货币政策委员会: 每季度一次
├── 全国金融工作会议: 约5年一次
├── 三中全会: 一般在换届后第二年秋季 (改革方向)
└── 行业专项会议: 不定期 (如全国住房城乡建设工作会议)
```

### 指数调整日历
```
沪深300/中证500等指数调整:
├── 调整频率: 每年2次
├── 调整时间: 6月第二个周五、12月第二个周五
├── 公告时间: 生效前约2周
├── 市场影响:
│   ├── 新纳入成分股: 被动资金买入 → 短期利好
│   ├── 被剔除成分股: 被动资金卖出 → 短期利空
│   └── 影响最大: 生效日前1-2周
└── 跟踪资金规模: 沪深300ETF约5000亿+

MSCI调整:
├── 调整频率: 每年4次 (2月/5月/8月/11月)
├── 季度调整 (小调): 比例微调
├── 半年度调整 (5月/11月大调): 可能新增/剔除成分股
└── 市场影响: 影响外资配置 → 北向资金流动

富时罗素调整:
├── 调整频率: 每年4次 (3月/6月/9月/12月)
└── 与MSCI类似，影响被动外资配置
```

## 三、交易限制相关日历

### 限售解禁
```
解禁类型:
├── IPO解禁:
│   ├── 网下配售: 上市后6个月解禁
│   ├── 控股股东/实控人: 上市后36个月解禁
│   ├── 其他股东: 上市后12个月解禁
│   └── 核心技术人员 (科创板): 12-48个月
│
├── 定增解禁:
│   ├── 竞价发行: 6个月 (2020年新规)
│   ├── 定价发行: 18个月
│   └── 控股股东/关联方: 18个月
│
└── 股权激励解禁:
    ├── 一般分批解锁: 12/24/36个月各解锁1/3
    └── 需满足业绩条件

解禁影响评估:
├── 解禁比例 > 流通盘20% → ⚠️ 显著供给压力
├── 解禁成本 < 现价50% → 减持动力强
├── 大股东解禁 + 质押率高 → ⚠️ 减持套现概率高
└── 解禁后: 观察大宗交易量 → 判断实际减持力度

月度解禁高峰:
├── 通常1月、6月、12月解禁规模较大
└── 关注wind/choice解禁日历提前规避
```

### 分红除权日历
```
A股分红季:
├── 年报分红: 4-7月 (年报披露后实施)
├── 中报分红: 少数公司，9-10月
├── 特别分红: 不定期
└── 港股通分红: 注意税率差异 (个人20%，企业10%或免)

分红相关日期:
├── 股权登记日: 持有该日股票可获分红
├── 除权除息日: 股价自动调整
├── 现金到账日: 通常除权后1-3个交易日
└── 注意: 持股期限影响个人所得税
    ├── 持有≤1个月: 税率20%
    ├── 1个月<持有≤1年: 税率10%
    └── 持有>1年: 免税

高股息策略日历:
├── 12-1月: 筛选高分红预期标的 (提前布局)
├── 3-4月: 年报分红方案公布 (确认)
├── 4-6月: 股权登记前买入/登记后卖出
└── 持续: 长期持有高股息策略不需择时
```

### 融资融券与两融
```
两融余额日历特征:
├── 节前: 通常去杠杆 (春节/国庆前)
├── 节后: 流动性恢复 (节后第一周)
├── 行情上涨期: 融资余额持续攀升
├── 行情下跌期: 融券卖出增加、融资余额下降
└── 极端值: 融资余额创新高/新低 → 情绪极端信号

两融标的调整:
├── 不定期调整 (通常随市场状况)
├── 新增两融标的 → 短期流动性改善
└── 被剔除两融标的 → ⚠️ 可能有风险
```

## 五、节假日与交易日

### 2026年A股休市安排 (参考往年规律)
```
预估休市安排 (以交易所正式公告为准):
├── 元旦: 1月1日-3日 (3天)
├── 春节: 约7天 (1月底-2月初)
├── 清明节: 4月4日-6日 (3天)
├── 劳动节: 5月1日-5日 (5天)
├── 端午节: 约3天 (6月)
├── 中秋节: 约3天 (9月)
├── 国庆节: 10月1日-7日 (7天)
└── 注意: 调休补班日也不交易

节假日效应:
├── 节前效应:
│   ├── 春节前: 通常偏弱 (资金回收/避险)
│   ├── 国庆前: 分化 (取决于政策预期)
│   └── 小长假前: 影响较小
│
├── 节后效应:
│   ├── 春节后: "春季躁动" 概率较高 (历史胜率>60%)
│   ├── 国庆后: 取决于假期消费数据和政策
│   └── 元旦后: "1月效应" (小盘股偏强)
│
└── 月末/季末效应:
    ├── 月末: 基金排名压力 → 部分抱团股波动
    ├── 季末: 银行冲存款 → 流动性偏紧
    └── 年末: 机构调仓 → 全年涨幅排名博弈
```

港股通特殊安排:

### 突发事件应对
```
常见突发事件类型:
├── 地缘政治: 贸易摩擦升级/军事冲突
│   └── 影响: 避险资产↑(黄金/国债)，风险资产↓
│
├── 监管政策: 突发行业监管政策
│   └── 影响: 相关行业剧烈波动，需快速评估影响范围
│
├── 流动性事件: 信用违约/银行风险
│   └── 影响: 信用利差走阔，高负债股下跌
│
├── 黑天鹅: 自然灾害/公共卫生事件
│   └── 影响: 市场恐慌性下跌，通常是中期买点
│
└── 技术性事件: 量化踩踏/ETF清盘
    └── 影响: 短期流动性冲击，通常快速修复

应对原则:
├── 先判断: 是否改变中长期逻辑?
├── 不恐慌: 多数突发事件影响是短期的
├── 看流动性: 流动性充裕时，冲击易修复
├── 等确认: 不在恐慌中做决策，等情况明朗
└── 逆向思考: "别人恐惧我贪婪" — 但需要确认基本面
```
