# 财务分析框架参考手册

## 一、杜邦分析 (DuPont Decomposition)

### 核心公式
```
ROE = 净利率 × 资产周转率 × 权益乘数

拆解:
ROE = (净利润/营收) × (营收/总资产) × (总资产/股东权益)
    = 净利率 × 周转率 × 杠杆

进一步拆解净利率:
净利率 = 毛利率 × (1 - 费用率 - 税率)
       = (营收-成本)/营收 × (1 - 销售费用率 - 管理费用率 - 研发费用率 - 财务费用率 - 税率)
```

### 杜邦分析三种驱动模型
| 驱动因素 | 典型行业 | 特征 |
|---------|---------|------|
| 高利润率驱动 | 白酒、奢侈品、软件 | 毛利率>60%，低周转 |
| 高周转率驱动 | 超市、快消、电商 | 薄利多销，高资产周转 |
| 高杠杆驱动 | 银行、地产 | 高负债率，放大收益 |

### 分析要点
1. ROE变动时，识别是哪个因素在变化
2. 同行业公司用相同框架比较
3. 警惕"靠杠杆提升ROE"的公司

## 二、利润表分析

### 营收分析
```
营收增长拆解:
├── 量增: 销售量增长 [X]%
├── 价增: 平均售价变化 [X]%
├── 结构变化: 高毛利产品占比变化
├── 新业务贡献: [X]%
└── 汇率影响: [X]% (如有海外业务)

营收质量检查:
├── 应收账款增速 vs 营收增速 (应收>营收 → ⚠️)
├── 预收款/合同负债变化 (增加 → 正面信号)
├── 关联方交易占比 (过高 → ⚠️)
└── 客户集中度 (前5大客户占比)
```

### 毛利率分析
```
毛利率变动因素:
├── 原材料成本变化
├── 产品价格变化
├── 产品结构变化 (高毛利产品占比)
├── 规模效应
├── 产能利用率变化
└── 汇率变动影响

毛利率趋势判断:
├── 连续3年上升 → 正面 (竞争力增强/结构优化)
├── 连续3年下降 → 负面 (竞争加剧/成本压力)
├── 波动大 → 周期性 (关注行业周期位置)
└── 稳定在高位 → 护城河可能较宽
```

### 费用率分析
```
三费率 = (销售费用 + 管理费用 + 财务费用) / 营收

各费用率关注点:
├── 销售费用率: 销售驱动型 vs 产品驱动型
│   ├── 消费品公司: 15-30% 正常
│   ├── B2B公司: 5-15% 正常
│   └── 趋势: 上升需关注是否有效果
│
├── 管理费用率: 管理效率
│   ├── 一般应随规模扩大而下降
│   └── 包含股权激励费用需单独分析
│
├── 研发费用率: 创新投入
│   ├── 科技公司: 10-25% 正常
│   ├── 关注: 研发资本化比例 (资本化率>50% → ⚠️)
│   └── 研发产出效率: 新产品营收/研发投入
│
└── 财务费用率: 融资成本
    ├── 净利息支出/总债务 = 实际融资成本
    └── 汇兑损益需单独考虑
```

### 净利润质量评估
```
净利润质量检查清单:
├── 扣非净利润 vs 净利润 (差异>20% → 关注非经常性项目)
├── 经营现金流/净利润 (>1.0 → 质量好; <0.7 → ⚠️)
├── 应收账款增速 vs 营收增速 (应收更快 → 利润可能虚高)
├── 存货增速 vs 营收增速 (存货积压风险)
├── 投资收益占比 (>20% → 利润可持续性存疑)
├── 公允价值变动 (波动大 → 利润不稳定)
└── 政府补贴占比 (>10% → 依赖性风险)
```

## 三、资产负债表分析

### 资产质量
```
资产质量检查:
├── 应收账款:
│   ├── 应收账款周转天数趋势
│   ├── 账龄结构 (1年以上占比)
│   ├── 坏账计提比例是否合理
│   └── 前5大应收账户集中度
│
├── 存货:
│   ├── 存货周转天数趋势
│   ├── 存货构成 (原材料/在产品/产成品)
│   ├── 跌价准备计提是否充分
│   └── 是否有过时存货风险
│
├── 商誉:
│   ├── 商誉/净资产比例 (>30% → 高风险)
│   ├── 被收购标的业绩承诺完成情况
│   ├── 减值测试假设是否激进
│   └── 历史减值记录
│
├── 固定资产:
│   ├── 在建工程占比 (大 → 扩产周期)
│   ├── 折旧政策是否合理
│   └── 资产利用率
│
└── 其他关注:
    ├── 其他应收款 (大额 → 可能隐藏关联交易)
    ├── 长期股权投资 (联营企业盈亏)
    └── 预付账款异常增加
```

### 负债结构
```
负债分析:
├── 总体杠杆:
│   ├── 资产负债率: [X]% (行业均值: [X]%)
│   ├── 有息负债率: [X]%
│   └── 净负债率: (有息负债-现金)/股东权益
│
├── 债务期限结构:
│   ├── 短期有息负债: [X]亿
│   ├── 长期有息负债: [X]亿
│   ├── 短期占比: [X]% (>70% → 再融资风险)
│   └── 近1年到期债务: [X]亿 vs 可用现金: [X]亿
│
├── 偿债能力:
│   ├── 流动比率: [X] (>1.5 安全)
│   ├── 速动比率: [X] (>1.0 安全)
│   ├── 利息覆盖倍数: [X] (>3.0 安全)
│   └── 经营现金流/有息负债: [X]%
│
└── 或有负债:
    ├── 担保金额
    ├── 未决诉讼
    └── 承诺与或有事项
```

## 四、现金流量表分析

### 经营现金流
```
经营现金流分析:
├── 净利润 → 经营现金流调节:
│   ├── + 折旧&摊销
│   ├── +/- 营运资本变动
│   │   ├── 应收账款变动
│   │   ├── 存货变动
│   │   └── 应付账款变动
│   └── +/- 其他非现金项
│
├── 自由现金流 (FCF):
│   FCF = 经营现金流 - 资本支出
│   └── FCF收益率 = FCF / 市值 (>5% → 有吸引力)
│
└── 现金流质量指标:
    ├── 经营现金流/净利润 (>1.0 → 好)
    ├── 经营现金流/营收 (稳定性)
    └── 自由现金流是否持续为正
```

### 投资现金流
```
投资活动分析:
├── 资本支出 (CapEx):
│   ├── CapEx/营收: [X]% (趋势: ↑/→/↓)
│   ├── CapEx/折旧: [X] (>1.5 → 扩张期; <1.0 → 维持/收缩)
│   └── CapEx用途: [扩产/维护/技改]
│
├── 收购活动:
│   ├── 金额: [X]亿
│   └── 收购标的及逻辑
│
└── 理财/投资:
    ├── 购买理财产品: [X]亿
    └── 投资收益: [X]亿
```

### 筹资现金流
```
筹资活动分析:
├── 借款: 新增 [X]亿, 偿还 [X]亿, 净增 [+/-X]亿
├── 发行股票: [X]亿 (如有)
├── 分红: [X]亿 (分红率: [X]%)
├── 回购: [X]亿 (如有)
└── 净筹资: [+/-X]亿

现金流综合判断:
├── 经营正+投资负+筹资负 → 优秀 (自我造血，回馈股东)
├── 经营正+投资负+筹资正 → 扩张 (借钱+赚钱一起投)
├── 经营正+投资正+筹资负 → 收缩 (出售资产还债)
├── 经营负+投资负+筹资正 → 危险 (烧钱+借钱)
└── 经营负+投资正+筹资正 → 极危险 (靠卖资产和借钱度日)
```

## 五、财务预警信号

```
红旗清单 (Red Flags):

🔴 严重警告:
├── 经营现金流连续2年为负
├── 商誉/净资产 > 50%
├── 应收账款增速远超营收增速 (>20%差距)
├── 审计意见非标准无保留
├── 大股东高比例质押 (>80%)
├── 频繁更换审计师
└── 存贷双高 (大量现金同时大量贷款)

🟡 需要关注:
├── 毛利率持续下滑
├── 研发资本化比例快速上升
├── 其他应收款/其他应付款异常增加
├── 关联交易规模扩大
├── 预付账款异常增加
├── 在建工程长期不转固定资产
└── 子公司或联营企业持续亏损
```

## 六、关键财务比率速查表

```
盈利能力:
├── 毛利率 = (营收-成本) / 营收
├── 净利率 = 净利润 / 营收
├── ROE = 净利润 / 股东权益
├── ROA = 净利润 / 总资产
├── ROIC = NOPAT / (股东权益+有息负债-现金)
└── 毛利率-净利率 差值 = 费用消耗率

成长性:
├── 营收增速 = (本期营收-上期营收) / 上期营收
├── 净利增速 = (本期净利-上期净利) / 上期净利
├── CAGR(N年) = (终值/初值)^(1/N) - 1
└── 内生增长率 = ROE × (1-分红率)

运营效率:
├── 应收周转天数 = 365 / (营收/平均应收)
├── 存货周转天数 = 365 / (成本/平均存货)
├── 应付周转天数 = 365 / (成本/平均应付)
├── 现金循环周期 = 应收天数 + 存货天数 - 应付天数
└── 总资产周转率 = 营收 / 总资产

偿债能力:
├── 流动比率 = 流动资产 / 流动负债
├── 速动比率 = (流动资产-存货) / 流动负债
├── 利息覆盖倍数 = EBIT / 利息支出
└── 净负债率 = (有息负债-现金) / 股东权益
```
