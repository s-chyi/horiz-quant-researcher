# 行业分析模板参考手册

## 一、行业生命周期定位

```
行业生命周期四阶段:

导入期 → 成长期 → 成熟期 → 衰退期
  │        │        │        │
  │        │        │        └── 特征: 需求萎缩、产能过剩、价格战
  │        │        │            投资: 回避或做空，关注转型企业
  │        │        │
  │        │        └── 特征: 增速放缓、集中度提升、分红增加
  │        │            投资: 龙头溢价，关注ROE和分红率
  │        │
  │        └── 特征: 高增速(20%+)、渗透率快速提升、竞争格局未定
  │            投资: 成长股逻辑，关注市占率和营收增速
  │
  └── 特征: 渗透率<5%、技术路线不明、高风险高回报
      投资: VC逻辑，小仓位博弈
```

### 渗透率与投资时机
```
渗透率关键节点:

0-5%     → 导入期 (技术验证阶段，不确定性极高)
5-15%    → 加速渗透期 ⭐ (最佳投资窗口，增速最快)
15-30%   → 快速普及期 (增速仍高，但竞争格局开始分化)
30-50%   → 成长后期 (增速放缓，龙头确立)
50-70%   → 成熟前期 (存量竞争为主)
>70%     → 成熟期 (替代需求+升级需求)

经典案例:
├── 智能手机: 2010年渗透率~10% → 加速期 → 2015年~70%成熟
├── 新能源车: 2020年渗透率~5% → 加速期 → 2025年~40%
├── 光伏: 2019年平价上网 → 加速渗透
└── AI大模型: 2024年企业渗透率~5% → 导入/加速期
```

## 二、行业竞争格局分析

### 波特五力模型
```
┌─────────────────────────────────────────┐
│             新进入者威胁                   │
│    (进入壁垒: 资金/技术/牌照/品牌/网络效应) │
└──────────────────┬──────────────────────┘
                   │
                   ▼
┌──────────┐  ┌──────────┐  ┌──────────┐
│ 供应商   │→ │ 行业竞争 │ ←│ 买方     │
│ 议价能力 │  │ 激烈程度 │  │ 议价能力 │
└──────────┘  └────┬─────┘  └──────────┘
                   │
                   ▼
┌─────────────────────────────────────────┐
│             替代品威胁                    │
│    (新技术/新模式对现有产品的替代)          │
└─────────────────────────────────────────┘
```

### 竞争格局分类
```
竞争格局类型:

垄断型 (CR1 > 50%):
├── 特征: 一家独大，定价权极强
├── 典型: 台积电(先进制程)、茅台(高端白酒)
└── 投资: 龙头享受垄断溢价

寡头型 (CR3 > 60%):
├── 特征: 3家以内主导，价格默契
├── 典型: 空调(格力+美的+海尔)、水泥(区域寡头)
└── 投资: 龙头稳定性高，关注份额变化

分散竞争型 (CR10 < 30%):
├── 特征: 行业碎片化，无明显龙头
├── 典型: 餐饮、装修、物流(部分)
└── 投资: 整合者机会，但风险也高

集中度提升期 (CR变化):
├── 特征: CR逐年提升，小企业退出
├── 典型: 啤酒、面板、快递
└── 投资: ⭐ 最佳投资阶段之一，龙头份额+利润双升
```

### 竞争格局量化指标
```
市场集中度指标:
├── CR1: 第一名市占率
├── CR3: 前三名合计市占率
├── CR5: 前五名合计市占率
├── CR10: 前十名合计市占率
└── HHI指数: Σ(市占率²) × 10000
    ├── HHI < 1000: 竞争性市场
    ├── 1000-1800: 中度集中
    └── HHI > 1800: 高度集中

竞争格局变化信号:
├── 正面: CR提升 + 龙头份额稳定/扩大 + 价格稳定
├── 负面: CR下降 + 新进入者增加 + 价格战
└── 转折: 政策变化 / 技术替代 / 新商业模式出现
```

## 三、产业链分析框架

### 产业链结构分析
```
标准产业链结构:

上游 (原材料/零部件)
│   └── 关注: 供给集中度、产能周期、价格弹性
│
├── 中游 (制造/加工)
│   └── 关注: 产能利用率、技术壁垒、规模效应
│
├── 下游 (品牌/渠道/终端)
│   └── 关注: 品牌溢价、渠道控制力、消费者黏性
│
└── 终端用户
    └── 关注: 需求趋势、价格敏感度、替代品风险
```

### 产业链利润分配
```
产业链微笑曲线:

  利润率
    │
    │  ★ 上游核心           ★ 下游品牌
    │   材料/IP                /渠道
    │     \                   /
    │      \                 /
    │       \    制造加工    /
    │        \  (利润最薄) /
    │         \          /
    │          \________/
    │
    └────────────────────────── 产业链环节

利润分配变化信号:
├── 上游涨价 → 中游毛利压缩 → 下游能否转嫁?
├── 下游砍价 → 中游被挤压 → 上游议价力如何?
├── 新技术出现 → 产业链利润重新分配
└── 政策干预 → 改变利润分配格局
```

### 产业链景气度传导
```
景气度传导顺序:

需求端景气:
终端需求↑ → 下游备货↑ → 中游订单↑ → 上游涨价
(传导时间: 1-3个月逐级传导)

衰退传导:
终端需求↓ → 下游去库存 → 中游砍单 → 上游跌价
(传导时间: 衰退传导通常更快)

库存周期放大效应 (牛鞭效应):
终端需求变化10% → 下游订单变化20% → 上游产量变化40%
└── 越上游波动越大，周期属性越强
```

## 四、行业空间测算

### 市场规模测算方法
```
方法一: 自上而下 (Top-Down)
总市场 = 目标人群 × 渗透率 × 客单价 × 使用频次
示例: 新能源车市场 = 年销量2800万辆 × 渗透率40% × 均价15万 = 1.68万亿

方法二: 自下而上 (Bottom-Up)
总市场 = Σ(各细分市场规模)
示例: 光伏市场 = 集中式 + 分布式(工商业+户用) + 海外

方法三: 类比法
参考成熟市场(如美国/日本)对应阶段的市场规模
调整系数: 人口比 × 收入比 × 消费习惯系数

方法四: 替代法
新产品市场 = 被替代产品的现有市场规模 × 替代比例
示例: 电动车市场 = 燃油车市场 × 替代渗透率
```

### 增量空间 vs 存量空间
```
行业空间分类:

增量市场 (市场还在扩大):
├── 新需求创造: 直播电商、AI应用
├── 替代需求: 新能源替代传统能源
├── 消费升级: 高端消费品
└── 测算重点: 渗透率 × 增速

存量市场 (市场规模稳定):
├── 份额争夺: 家电、日化
├── 升级换代: 手机、汽车
├── 区域扩张: 区域连锁→全国
└── 测算重点: 集中度提升空间 × 龙头份额增量

关键判断:
├── 增量市场: 行业β > 个股α → 选赛道
├── 存量市场: 个股α > 行业β → 选龙头
└── 转折点: 行业从增量转存量时，估值逻辑切换
```

## 五、行业政策分析

### 政策分析框架
```
政策分析四维度:

1. 政策方向:
   ├── 鼓励类: 补贴、减税、融资支持
   ├── 限制类: 限产、环保标准、反垄断
   ├── 规范类: 牌照管理、标准制定、数据安全
   └── 中性类: 行业标准更新、技术路线指引

2. 政策力度:
   ├── 国务院级: 顶层设计，影响深远
   ├── 部委级: 具体执行方案
   ├── 地方级: 区域性影响
   └── 行业自律: 影响较小

3. 政策时效:
   ├── 短期刺激: 1-2个季度 (如消费券)
   ├── 中期影响: 1-3年 (如补贴政策)
   └── 长期重塑: 3年+ (如碳达峰、共同富裕)

4. 执行节奏:
   ├── 文件下发 → 细则出台 → 地方落地 → 效果显现
   └── 时滞通常: 3-12个月
```

### 政策敏感行业清单
```
高政策敏感度:
├── 房地产: 限购/限贷/利率/土地政策
├── 医药: 集采/医保/创新药审批
├── 教育: 双减/民办学校政策
├── 互联网: 反垄断/数据安全/平台经济
├── 金融: 资管新规/利率政策/监管要求
└── 新能源: 补贴退坡/碳交易/储能政策

中政策敏感度:
├── 消费: 消费刺激政策/关税
├── 制造业: 智能制造/专精特新
├── 基建: 财政政策/专项债
└── 农业: 粮食安全/种业政策

低政策敏感度:
├── 必选消费: 食品/日用品
├── 公用事业: 水电燃气(但价格受管制)
└── 部分ToB软件: 受宏观政策影响较小
```

## 六、行业比较维度

### 行业吸引力评估矩阵
```
评估维度 (每项1-10分):

1. 市场空间: 行业规模和增长空间
   ├── 10分: 万亿级市场，年增20%+
   ├── 7分: 千亿级市场，年增10-20%
   ├── 5分: 百亿级市场，年增5-10%
   └── 3分: 小市场或增速<5%

2. 竞争格局: 行业集中度和壁垒
   ├── 10分: 寡头垄断，壁垒极高
   ├── 7分: 集中度高，龙头地位稳固
   ├── 5分: 集中度中等，竞争可控
   └── 3分: 高度分散，价格战频繁

3. 盈利能力: ROE和利润率水平
   ├── 10分: ROE>20%，毛利率>50%
   ├── 7分: ROE 15-20%，毛利率30-50%
   ├── 5分: ROE 10-15%，毛利率20-30%
   └── 3分: ROE<10%，毛利率<20%

4. 成长性: 营收和利润增速
   ├── 10分: 营收增速>30%
   ├── 7分: 营收增速15-30%
   ├── 5分: 营收增速5-15%
   └── 3分: 营收增速<5%

5. 政策风险: 政策不确定性
   ├── 10分: 政策大力扶持，确定性高
   ├── 7分: 政策中性，行业自主发展
   ├── 5分: 有政策风险但可预测
   └── 3分: 政策风险高，方向不明

综合评分 = Σ(维度分数 × 权重)
├── 默认权重: 各20%
└── 可根据投资风格调整权重
```

### 行业轮动参考
```
经济周期与行业轮动:

      经济过热           经济衰退
         │                 │
    能源/材料          医药/公用事业
    有色/化工          食品饮料/电力
         │                 │
         ▼                 ▼
    ←─── 顺时针轮动 ────→
         ▲                 ▲
         │                 │
    金融/地产           科技/成长
    消费/制造           军工/新能源
         │                 │
      经济复苏           经济滞胀

A股行业轮动经验:
├── 流动性宽松初期: 券商 → 科技成长
├── 经济复苏确认: 消费 → 周期
├── 经济过热: 上游资源 → 银行
├── 紧缩周期: 防御(公用/医药/消费)
└── 注意: A股受政策影响大，纯周期轮动不完全适用
```

## 七、重点行业分析模板

### 消费品行业
```
核心关注:
├── 品牌力: 提价能力、消费者心智
├── 渠道力: 渠道覆盖、终端掌控力
├── 产品力: 产品矩阵、新品成功率
├── 增长驱动: 量增 vs 价增 vs 结构升级
└── 关键指标: 同店增速、渠道库存、经销商数量
```

### 科技/制造行业
```
核心关注:
├── 技术壁垒: 专利数量、研发投入、技术代差
├── 产能周期: 产能利用率、在建产能、扩产计划
├── 下游需求: 订单能见度、客户结构
├── 国产替代: 替代空间、替代进度
└── 关键指标: 研发占比、产能利用率、订单增速、良率
```

### 周期行业
```
核心关注:
├── 供需缺口: 产能/产量 vs 需求量
├── 库存周期: 当前库存位置(高位/低位)
├── 价格弹性: 价格对盈利的敏感度
├── 成本曲线: 边际成本、成本分位
└── 关键指标: 产品价格、库存天数、产能利用率、开工率

周期股投资要点:
├── 买入: 亏损扩大 + 产能出清 + 库存见底 → PE最高时
├── 卖出: 盈利创新高 + 产能扩张 + 库存堆积 → PE最低时
└── 核心: 周期股"低PE不是买点，高PE不是卖点"
```

### 金融行业
```
银行核心关注:
├── 息差: NIM(净息差)趋势
├── 资产质量: 不良率、关注类、拨备覆盖率
├── 规模增长: 贷款增速、存款增速
├── 资本充足: 核心一级资本充足率
└── 关键指标: NIM、不良率、拨备覆盖率、ROE

保险核心关注:
├── 负债端: NBV(新业务价值)增速、代理人数量
├── 资产端: 投资收益率、权益配置
├── 内含价值: EV增速、运营偏差
└── 关键指标: NBV增速、综合成本率、投资收益率
```
