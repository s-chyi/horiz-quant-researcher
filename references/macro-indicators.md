# 宏观指标体系参考手册

## 一、中国核心宏观指标

### 经济增长指标
```
GDP (国内生产总值):
├── 发布: 国家统计局，季度发布 (初步核算/最终核实)
├── 关注: 实际GDP增速、名义GDP增速
├── 当前目标: 5%左右 (2024-2025)
├── 领先/滞后: 滞后指标
└── 投资含义:
    ├── GDP超预期 → 顺周期板块(金融/地产/消费)受益
    └── GDP不及预期 → 政策宽松预期升温 → 利好成长/科技

工业增加值:
├── 发布: 国家统计局，月度发布 (规模以上)
├── 关注: 同比增速、环比增速
├── 领先/滞后: 同步指标
└── 投资含义: 反映制造业景气度，影响工业/周期板块

固定资产投资:
├── 发布: 国家统计局，月度累计同比
├── 关注: 制造业/房地产/基建 三大分项
├── 领先/滞后: 同步指标
└── 投资含义:
    ├── 基建投资↑ → 建筑/建材/工程机械
    ├── 房地产投资↑ → 地产链(钢铁/水泥/家电)
    └── 制造业投资↑ → 设备/自动化/工业软件

社会消费品零售总额:
├── 发布: 国家统计局，月度发布
├── 关注: 同比增速(名义/实际)、限额以上分品类
├── 领先/滞后: 同步指标
└── 投资含义: 反映消费景气，影响消费板块
```

### PMI (采购经理指数)
```
制造业PMI:
├── 发布: 国家统计局 + 财新 (每月1日左右)
├── 枯荣线: 50
│   ├── >50: 扩张
│   ├── <50: 收缩
│   └── 趋势比绝对值更重要
├── 分项关注:
│   ├── 新订单: 需求端先行指标
│   ├── 新出口订单: 外需风向标
│   ├── 生产: 供给端活跃度
│   ├── 原材料库存: 补库/去库意愿
│   ├── 产成品库存: 库存周期判断
│   └── 价格(购进/出厂): 通胀先行
├── 领先/滞后: 领先指标 (领先经济1-2个月)
└── 投资含义:
    ├── PMI连续上行 → 经济复苏确认 → 顺周期
    ├── PMI见顶回落 → 经济放缓预警 → 防御切换
    └── 官方PMI vs 财新PMI 分歧 → 大企业vs中小企业景气分化

非制造业PMI:
├── 服务业PMI: 消费和服务业景气
└── 建筑业PMI: 基建和地产施工景气
```

### 通胀指标
```
CPI (居民消费价格指数):
├── 发布: 国家统计局，月度发布
├── 关注:
│   ├── CPI同比: 整体通胀水平
│   ├── 核心CPI: 剔除食品和能源 (更能反映需求)
│   ├── 食品CPI: 猪肉价格影响大
│   └── 非食品CPI: 反映服务业需求
├── 关键阈值:
│   ├── CPI > 3%: 通胀压力，货币政策可能收紧
│   ├── CPI 1-3%: 温和通胀，经济健康
│   ├── CPI 0-1%: 需求偏弱，关注通缩风险
│   └── CPI < 0%: 通缩，需要积极政策刺激
└── 投资含义:
    ├── 温和通胀 → 消费品提价逻辑 → 利好品牌消费
    ├── 高通胀 → 上游涨价 → 利好资源品
    └── 通缩 → 政策宽松 → 利好债市和成长股

PPI (工业生产者出厂价格指数):
├── 发布: 国家统计局，月度发布
├── 关注:
│   ├── PPI同比: 工业品价格总水平
│   ├── PPI环比: 价格变化方向
│   ├── 生产资料 vs 生活资料: 结构分化
│   └── PPI-CPI剪刀差: 中下游利润空间
├── 领先/滞后: 与大宗商品价格同步
└── 投资含义:
    ├── PPI上行 → 上游资源品盈利改善
    ├── PPI下行 → 中下游成本压力缓解
    └── PPI-CPI剪刀差缩小 → 中下游利润修复
```

### 货币与信用指标
```
M2 (广义货币):
├── 发布: 央行，月度发布
├── 关注: M2同比增速、M2-M1剪刀差
├── M2-M1剪刀差:
│   ├── 剪刀差缩小 → 资金活化，经济活跃度提升
│   └── 剪刀差扩大 → 资金定期化，经济活力不足
└── 投资含义: M2持续高增 → 流动性充裕 → 利好权益市场

社会融资规模:
├── 发布: 央行，月度发布
├── 关注:
│   ├── 新增社融总量: 整体信用扩张/收缩
│   ├── 人民币贷款: 实体经济融资需求
│   ├── 企业债券: 直接融资渠道
│   ├── 政府债券: 财政发力程度
│   └── 社融存量增速: 趋势性变化
├── 领先/滞后: 领先经济2-3个季度
└── 投资含义:
    ├── 社融放量 → 经济企稳预期 → 利好顺周期
    ├── 社融收缩 → 信用紧缩 → 利空高杠杆行业
    └── "宽信用"拐点 → 股市见底的重要信号

信贷结构:
├── 中长期贷款: 反映实体投资意愿
│   ├── 企业中长期贷款↑ → 企业投资信心恢复
│   └── 居民中长期贷款↑ → 房贷需求 → 地产回暖
├── 短期贷款/票据融资:
│   └── 占比过高 → "冲量"嫌疑，实际需求不强
└── 信贷脉冲: 新增信贷/GDP变化率 → 领先经济
```

### 利率与流动性
```
政策利率体系:
├── LPR (贷款市场报价利率):
│   ├── 1年期LPR: 影响企业贷款、消费贷
│   ├── 5年期LPR: 影响房贷利率
│   └── 调整含义: 降LPR → 降低实体融资成本
│
├── MLF利率 (中期借贷便利):
│   ├── 目前1年期，每月15日左右操作
│   └── LPR的锚，MLF降 → LPR大概率跟降
│
├── OMO利率 (7天逆回购):
│   ├── 短端利率锚
│   └── 调整频率最高的政策利率
│
└── 存款准备金率 (RRR):
    ├── 降准0.25% ≈ 释放约5000亿长期资金
    ├── 降准0.5% ≈ 释放约1万亿长期资金
    └── 降准 → 银行可贷资金增加 → 利好银行/地产

市场利率:
├── DR007 (银行间7天回购加权利率):
│   ├── 反映银行间流动性松紧
│   ├── < 政策利率: 流动性宽松
│   └── > 政策利率: 流动性偏紧
│
├── 10年期国债收益率:
│   ├── 无风险利率的基准
│   ├── 下行 → 利好成长股估值 (分母逻辑)
│   ├── 上行 → 估值承压，利好银行
│   └── 与美债利差 → 影响资金流向和汇率
│
├── 信用利差:
│   ├── AA+与国债利差: 信用风险定价
│   ├── 利差走阔 → 信用风险上升 → 利空高负债企业
│   └── 利差收窄 → 信用环境改善
│
└── Shibor (上海银行间同业拆放利率):
    └── 3个月Shibor: 银行资金成本指标
```

### 房地产指标
```
房地产核心数据:
├── 销售端:
│   ├── 商品房销售面积/金额: 需求端核心
│   ├── 30城成交面积: 高频跟踪 (日/周)
│   ├── 百城房价指数: 价格趋势
│   └── 二手房挂牌量/成交量: 真实需求
│
├── 投资端:
│   ├── 房地产开发投资: 行业投资总量
│   ├── 新开工面积: 前瞻指标
│   ├── 施工面积: 在建规模
│   ├── 竣工面积: 竣工周期
│   └── 土地购置面积/金额: 拿地意愿
│
├── 资金端:
│   ├── 房企到位资金: 现金流状况
│   ├── 居民中长期贷款: 按揭需求
│   └── 房企融资渠道: 信贷/债券/信托
│
└── 传导链:
    销售回暖 → 拿地恢复 → 新开工增加 → 施工/竣工
    (每个环节传导约3-6个月)

房地产产业链影响:
├── 上游: 钢铁、水泥、玻璃、铝材
├── 中游: 建筑施工、工程机械
├── 下游: 装修、家电、家具
└── 金融: 银行房贷、地产债
```

### 外贸与汇率
```
进出口数据:
├── 发布: 海关总署，月度发布 (美元/人民币计价)
├── 关注:
│   ├── 出口同比: 外需强弱
│   ├── 进口同比: 内需强弱 + 大宗商品进口
│   ├── 贸易顺差: 经常账户收支
│   └── 按国家/商品分项: 结构性变化
└── 投资含义:
    ├── 出口超预期 → 出口导向型企业受益
    ├── 进口大宗商品增加 → 内需恢复信号
    └── 贸易摩擦升级 → 避险情绪

人民币汇率:
├── USD/CNY 在岸汇率
├── USD/CNH 离岸汇率
├── CFETS人民币汇率指数 (一篮子货币)
├── 中间价 vs 即期汇率 (偏离度反映央行态度)
└── 投资含义:
    ├── 人民币升值 → 利好进口型企业、航空、造纸
    ├── 人民币贬值 → 利好出口型企业、纺织、家电
    ├── 贬值预期 → 北向资金可能流出
    └── 升值预期 → 吸引外资流入
```

## 二、美国核心宏观指标

### 就业数据
```
非农就业 (Nonfarm Payrolls):
├── 发布: 劳工统计局BLS，每月第一个周五
├── 关注: 新增就业人数、失业率、时薪增速
├── 对市场影响: ⭐⭐⭐⭐⭐ (最重要的月度数据之一)
└── 传导:
    ├── 非农强劲 → 加息预期升温 → 美元↑ 美债↑ 美股↓(短期)
    └── 非农疲弱 → 降息预期升温 → 美元↓ 美债↓ 美股↑(短期)

失业率:
├── 自然失业率约4.0-4.5%
├── <4.0%: 劳动力市场紧张 → 通胀压力
└── >5.0%: 经济放缓信号

初请失业金人数 (Weekly Jobless Claims):
├── 发布: 每周四
├── 高频数据，可提前预判非农趋势
└── 持续上升 → 就业市场恶化
```

### 通胀数据
```
CPI:
├── 发布: BLS，月中发布
├── 关注: 总体CPI、核心CPI (剔除食品能源)
├── Fed目标: 2% (PCE口径)
└── 对市场影响: ⭐⭐⭐⭐⭐

PCE (个人消费支出价格指数):
├── 发布: 经济分析局BEA，月末发布
├── Fed更关注PCE而非CPI
├── 核心PCE: Fed通胀目标的直接衡量
└── 对市场影响: ⭐⭐⭐⭐

传导至A股:
├── 美国通胀高企 → Fed加息 → 美债收益率↑ → 中美利差变化
│   → 人民币贬值压力 → 北向资金流出 → A股成长股承压
└── 美国通胀回落 → Fed降息预期 → 全球风险偏好改善
    → 新兴市场资金回流 → A股受益
```

## 三、库存周期

### 库存周期四阶段
```
库存周期 (基钦周期，约3-4年一轮):

       需求↑              需求↓
    ┌──────────┐      ┌──────────┐
    │ 被动去库 │ →→→ │ 主动补库 │
    │ 需求↑   │      │ 需求↑   │
    │ 库存↓   │      │ 库存↑   │
    │ 最佳买点 │      │ 经济繁荣 │
    └──────────┘      └──────────┘
         ↑                  │
         │                  ▼
    ┌──────────┐      ┌──────────┐
    │ 主动去库 │ ←←← │ 被动补库 │
    │ 需求↓   │      │ 需求↓   │
    │ 库存↓   │      │ 库存↑   │
    │ 经济萧条 │      │ 经济放缓 │
    └──────────┘      └──────────┘

各阶段投资策略:
├── 被动去库 → ⭐ 最佳买点 → 做多周期股/顺周期
├── 主动补库 → 持有期 → 周期股盈利兑现
├── 被动补库 → 卖出信号 → 减持周期，转向防御
└── 主动去库 → 回避期 → 持有防御品种，等待拐点
```

### 库存周期判断指标
```
库存周期判断:
├── 工业企业产成品存货同比
├── PMI产成品库存/原材料库存
├── 行业库存数据 (如钢材社会库存、化工库存)
├── 补库/去库意愿: PMI采购量 vs 产成品库存
└── 价格验证: PPI是否配合

当前周期位置判断方法:
├── 产成品存货同比 + 营收同比:
│   ├── 营收↑ + 存货↓ → 被动去库 (⭐买点)
│   ├── 营收↑ + 存货↑ → 主动补库 (持有)
│   ├── 营收↓ + 存货↑ → 被动补库 (卖点)
│   └── 营收↓ + 存货↓ → 主动去库 (观望)
└── 注意: 不同行业库存周期不同步
```

## 四、宏观-行业传导链

### 利率传导
```
降息传导链:

央行降息
├── 第一阶: 资金面 (即时)
│   ├── 银行间利率下降
│   ├── 债券收益率下行 → 债市上涨
│   └── 贴现率下降 → 股市估值提升
│
├── 第二阶: 信贷 (1-3个月)
│   ├── 贷款利率下降 → 企业融资成本降低
│   ├── 按揭利率下降 → 房贷压力减轻
│   └── 信用扩张 → 社融回升
│
├── 第三阶: 实体经济 (3-6个月)
│   ├── 投资意愿恢复
│   ├── 消费信心改善
│   └── 房地产销售回暖
│
└── 受益行业排序:
    ├── 立即受益: 券商、保险 (权益资产升值)
    ├── 短期受益: 银行 (但息差可能缩小)、地产
    ├── 中期受益: 高负债行业 (地产/基建)
    └── 长期受益: 成长股 (折现率下降 → 估值上升)
```

### 财政政策传导
```
财政扩张传导链:

专项债/基建投资加码
├── 第一阶: 直接受益 (立即)
│   ├── 建筑/工程: 订单增加
│   ├── 建材: 水泥/钢材需求
│   └── 工程机械: 设备需求
│
├── 第二阶: 间接受益 (1-3个月)
│   ├── 上游: 钢铁/有色需求
│   ├── 就业: 建筑工人收入增加
│   └── 区域经济: 基建带动地方经济
│
├── 第三阶: 乘数效应 (3-6个月)
│   ├── 消费改善: 就业→收入→消费
│   └── 投资带动: 基建→物流→产业
│
└── 注意事项:
    ├── 财政乘数: 中国约0.6-1.0 (基建投资)
    ├── 挤出效应: 政府举债→利率上升→民间投资下降
    └── 效率递减: 重复建设→投资效率下降
```

### 汇率传导
```
人民币贬值传导链:

人民币贬值
├── 第一阶: 资金面 (即时)
│   ├── 北向资金流出压力
│   ├── 外币资产升值 (航空/造纸等美元负债↓)
│   └── 进口成本上升
│
├── 第二阶: 贸易 (1-3个月)
│   ├── 出口竞争力提升 → 出口型企业受益
│   ├── 进口成本上升 → 进口替代企业受益
│   └── 大宗商品(进口): 人民币计价上涨
│
├── 第三阶: 预期 (3-6个月)
│   ├── 贬值预期 → 资本外流压力
│   ├── 央行可能干预 → 外汇储备变化
│   └── 可能限制货币政策宽松空间
│
└── 受益/受损行业:
    ├── 受益: 纺织服装、家电、机械(出口型)
    ├── 受损: 航空(美元负债)、造纸(进口纸浆)
    └── 复杂: 石油化工(进口原料但产品内销)
```

## 六、宏观研判框架

### 经济状态判断
```
综合研判矩阵:

        增长↑              增长↓
     ┌──────────┐      ┌──────────┐
通胀↑│  过热     │      │  滞胀     │
     │ 紧缩政策 │      │ 最差组合 │
     │ 商品/资源│      │ 现金/黄金│
     └──────────┘      └──────────┘
     ┌──────────┐      ┌──────────┐
通胀↓│  复苏     │      │  衰退     │
     │ 最优组合 │      │ 宽松政策 │
     │ 股票/周期│      │ 债券/防御│
     └──────────┘      └──────────┘

判断依据:
├── 增长: GDP、PMI、工业增加值、社融
├── 通胀: CPI、PPI、核心通胀
├── 政策: 利率方向、财政力度
└── 外部: 美联储政策、全球需求
```

### 政策预判方法
```
政策预判三步法:

第一步: 经济目标 vs 现实差距
├── GDP目标 vs 实际增速
├── 就业目标 vs 失业率
└── 差距大 → 政策加码概率高

第二步: 政策空间评估
├── 降息空间: 当前利率 vs 历史低点
├── 降准空间: 当前准备金率 vs 最低水平
├── 财政空间: 赤字率 vs 3%红线
└── 空间大 → 政策力度可能更强

第三步: 政策信号捕捉
├── 国务院常务会议: 政策方向的高频信号
├── 央行操作: OMO/MLF的量和价
├── 政策吹风: 官媒/智库文章
└── 国际协调: 中美高层对话/G20
```
