# 风险评估方法论参考手册

## 一、风险分类体系

```
投资风险四大类:

├── 1. 基本面风险 (Fundamental Risk)
│   ├── 财务风险: 资产质量恶化、现金流断裂、偿债困难
│   ├── 经营风险: 核心业务下滑、竞争力减弱、管理层变动
│   ├── 治理风险: 大股东掏空、关联交易、信息披露问题
│   └── 成长风险: 增速不及预期、新业务失败、转型受阻
│
├── 2. 市场风险 (Market Risk)
│   ├── 估值风险: 估值过高、泡沫化
│   ├── 流动性风险: 成交量不足、大宗交易折价
│   ├── 风格切换: 成长→价值、大盘→小盘
│   └── 情绪风险: 恐慌性抛售、非理性杀跌
│
├── 3. 外部风险 (External Risk)
│   ├── 政策风险: 行业监管、税收变化、产业政策
│   ├── 宏观风险: 经济衰退、利率变化、汇率波动
│   ├── 地缘风险: 贸易摩擦、制裁、地缘冲突
│   └── 技术替代: 颠覆性技术出现
│
└── 4. 事件风险 (Event Risk)
    ├── 黑天鹅: 突发事件(疫情、自然灾害)
    ├── 诉讼风险: 重大法律纠纷
    ├── 安全事故: 产品质量/安全生产问题
    └── 信用事件: 债务违约、评级下调
```

## 二、概率-影响矩阵

### 标准矩阵模板
```
影响程度
  │
极大│  ③中等    ④高风险    ⑤极高
(>30%)│
  │
  大 │  ②中低    ③中等     ④高风险
(15-30%)│
  │
 中等│  ①低风险   ②中低     ③中等
(5-15%)│
  │
  小 │  ①低风险   ①低风险    ②中低
(<5%) │
  │
  └──────┬──────┬──────┬──────
        低      中      高
      (<20%)  (20-60%) (>60%)
              发生概率

风险等级对应行动:
├── ⑤ 极高风险: 立即行动，考虑清仓或大幅减仓
├── ④ 高风险: 制定应急预案，设置止损位
├── ③ 中等风险: 密切监控，控制仓位
├── ② 中低风险: 定期检查，纳入观察
└── ① 低风险: 常规监控即可
```

### 影响程度量化标准
```
影响程度评估标准:

极大 (对股价影响 > 30%):
├── 核心业务面临生存威胁
├── 财务造假或重大违规
├── 行业被政策彻底颠覆 (如教培双减)
├── 核心产品/技术被完全替代
└── 流动性危机导致债务违约

大 (对股价影响 15-30%):
├── 核心业务增速大幅低于预期
├── 重要客户/订单流失
├── 大股东/管理层重大变动
├── 商誉大额减值
└── 行业竞争格局恶化

中等 (对股价影响 5-15%):
├── 单季度业绩不及预期
├── 原材料成本大幅上涨
├── 次要业务出现问题
├── 高管减持 (非控股股东)
└── 行业政策微调

小 (对股价影响 < 5%):
├── 短期经营波动
├── 非核心子公司问题
├── 市场情绪波动
└── 分析师下调评级
```

### 发生概率评估标准
```
概率评估依据:

高概率 (>60%):
├── 已有明确信号 (如政策已发布征求意见稿)
├── 历史上频繁发生
├── 行业趋势已经确立
└── 财务指标已出现恶化迹象

中概率 (20-60%):
├── 有一定信号但不确定
├── 行业存在此类风险先例
├── 多数分析师已关注
└── 需要特定条件触发

低概率 (<20%):
├── 属于极端情景
├── 历史上罕见
├── 需要多个负面因素同时出现
└── 当前无明显信号
```

## 三、个股风险扫描清单

### 财务健康度检查
```
🔴 严重风险信号 (任一触发即需深入调查):

1. 现金流:
   ├── 经营现金流连续2年为负
   ├── 经营现金流/净利润 < 0.5 (连续2年)
   └── 自由现金流连续3年为负

2. 资产质量:
   ├── 商誉/净资产 > 50%
   ├── 应收账款增速 > 营收增速 + 20%
   ├── 存货增速 > 营收增速 + 30%
   └── 其他应收款突然大幅增加

3. 债务:
   ├── 短期有息负债 > 现金 + 经营现金流
   ├── 利息覆盖倍数 < 1.5
   ├── 资产负债率 > 行业均值 + 20%
   └── 存贷双高 (大量现金同时大量贷款)

4. 审计/合规:
   ├── 非标审计意见
   ├── 频繁更换审计师
   ├── 会计政策频繁变更
   └── 内部控制存在重大缺陷

5. 大股东:
   ├── 大股东质押率 > 80%
   ├── 大股东/董监高大额减持
   ├── 大额关联交易
   └── 控制权之争
```

### 经营风险检查
```
🟡 需要关注的经营信号:

1. 业绩质量:
   ├── 扣非净利润/净利润 < 70%
   ├── 政府补贴占净利润 > 20%
   ├── 投资收益占净利润 > 30%
   └── 公允价值变动损益波动大

2. 竞争力变化:
   ├── 毛利率连续3个季度下滑
   ├── 市占率下降
   ├── 核心客户流失
   ├── 研发投入下降
   └── 核心人才流失

3. 运营效率:
   ├── 应收周转天数持续上升
   ├── 存货周转天数持续上升
   ├── 产能利用率下降
   └── 在建工程长期不转固

4. 增长持续性:
   ├── 营收增速连续放缓
   ├── 订单/合同负债下滑
   ├── 新产品/新市场进展不及预期
   └── 渗透率接近天花板
```

### 治理风险检查
```
⚠️ 公司治理红旗:

1. 股权结构:
   ├── 实控人一股独大且缺乏制衡
   ├── 股权结构复杂、交叉持股多
   ├── 代持、信托持股等不透明安排
   └── VIE架构风险 (海外上市公司)

2. 管理层:
   ├── 核心管理层频繁变动
   ├── 管理层薪酬与业绩严重不匹配
   ├── 管理层有不良记录
   └── 董事会独立性不足

3. 信息披露:
   ├── 信息披露质量差或模糊
   ├── 多次收到交易所问询函/关注函
   ├── 延迟披露重大事项
   └── 前后表述不一致

4. 资本运作:
   ├── 频繁并购且标的质量存疑
   ├── 高溢价收购关联方资产
   ├── 业绩承诺到期后标的业绩大幅下滑
   └── 频繁定增但不分红
```

## 四、组合风险评估

### 集中度风险
```
组合集中度检查:

单一股票集中度:
├── 单只个股 > 30% → ⚠️ 高集中度风险
├── 单只个股 > 20% → 🟡 建议关注
└── 单只个股 ≤ 10% → ✅ 合理分散

行业集中度:
├── 单一行业 > 40% → ⚠️ 行业风险过度暴露
├── 单一行业 > 25% → 🟡 建议关注
└── 单一行业 ≤ 15% → ✅ 行业分散

风格集中度:
├── 全部成长股 → ⚠️ 风格切换风险
├── 全部价值股 → ⚠️ 可能错过成长机会
└── 成长+价值+防御 → ✅ 风格平衡

市值集中度:
├── 全部大盘股 → 弹性不足
├── 全部小盘股 → 流动性风险
└── 大+中+小搭配 → ✅ 合理
```

### 相关性风险
```
组合相关性分析:

高相关性风险场景:
├── 同一产业链上下游 (如锂矿+锂电池+新能源车)
│   └── 需求下滑时全线下跌
├── 同类因子暴露 (如全部高PE成长股)
│   └── 风格切换时一起杀估值
├── 同一宏观因子驱动 (如全部出口型企业)
│   └── 汇率/贸易政策变化时同向波动
└── 同一大股东/关联方
    └── 资金链风险传染

降低相关性的方法:
├── 跨行业配置 (至少3个不相关行业)
├── 跨风格配置 (成长+价值+防御)
├── 跨市值配置 (大盘+中盘+小盘)
├── 跨地域配置 (内需+出口; A股+港股)
└── 跨周期配置 (早周期+晚周期+逆周期)
```

极端情景分析:

## 五、风险量化指标

### 波动性指标
```
波动率相关:
├── 历史波动率: 过去N日收益率的标准差 × √252
│   ├── A股个股年化波动率典型区间: 30-60%
│   └── 沪深300年化波动率典型区间: 15-25%
│
├── Beta系数: 个股波动 / 市场波动
│   ├── β > 1.5: 高弹性，涨跌都放大
│   ├── β ≈ 1.0: 与市场同步
│   └── β < 0.7: 防御性，涨跌幅小于市场
│
└── 最大回撤 (Max Drawdown):
    ├── = (峰值 - 谷值) / 峰值
    ├── A股个股典型最大回撤: 30-70%
    └── 组合最大回撤控制目标: <20%(保守) / <30%(平衡) / <40%(激进)
```

### 下行风险指标
```
下行风险度量:
├── VaR (在险价值):
│   ├── 95% VaR: 正常市场下，5%概率的最大损失
│   ├── 99% VaR: 极端情况下，1%概率的最大损失
│   └── 计算: 历史模拟法 / 参数法 / Monte Carlo
│
├── CVaR (条件在险价值):
│   └── = 超过VaR之后的平均损失 (比VaR更保守)
│
├── 下行波动率:
│   └── 只计算负收益日的波动率 (Sortino Ratio用)
│
└── 回撤恢复期:
    ├── 从最大回撤恢复到前高所需时间
    └── A股大熊市回撤恢复期: 通常2-5年
```

## 六、风险应对策略

### 事前风控 (预防)
```
投资前风控清单:
├── 仓位管理:
│   ├── 单只个股最大仓位: [X]%
│   ├── 单行业最大仓位: [X]%
│   ├── 总仓位上限: [X]%
│   └── 新建仓位不超过: [X]%
│
├── 买入条件检查:
│   ├── 基本面: 满足选股标准?
│   ├── 估值: 在合理区间内?
│   ├── 技术面: 没有明显破位信号?
│   ├── 资金面: 没有大规模流出?
│   └── 风险: 已识别的风险可接受?
│
└── 止损预设:
    ├── 硬止损: 亏损 [X]% 无条件止损
    ├── 软止损: 基本面恶化时止损
    └── 时间止损: [X]个月未达预期时重新评估
```

### 事中风控 (监控)
```
持仓期间监控:

日频监控:
├── 股价异常波动 (单日涨跌>5%)
├── 成交量异常放大 (>5日均量2倍)
├── 重要公告/新闻
└── 板块/行业资金流向

周频监控:
├── 北向资金持仓变化
├── 融资融券余额变化
├── 行业景气度数据
└── 竞争对手动态

季频监控:
├── 财报数据 vs 预期
├── 核心经营指标变化
├── 估值分位数变化
└── 组合整体风险暴露

不定期关注:
├── 管理层/大股东动态
├── 政策变化
├── 行业突发事件
└── 宏观环境变化
```

### 事后风控 (应对)
```
风险事件应对决策树:

风险事件发生
├── 判断: 是否影响核心投资逻辑?
│   ├── 是 → 重新评估
│   │   ├── 逻辑已破坏 → 果断止损
│   │   └── 逻辑受损但可修复 → 减仓观察
│   │
│   └── 否 → 判断: 是短期冲击还是长期影响?
│       ├── 短期冲击 → 持有或加仓
│       │   └── 条件: 基本面未变 + 估值更便宜
│       └── 长期影响 → 减仓或止损
│
├── 止损执行原则:
│   ├── 到达止损位 → 先执行，再反思
│   ├── 不要"补仓摊薄成本"来逃避止损
│   └── 止损后可以重新买入，但需要新的理由
│
└── 复盘总结:
    ├── 风险信号是否可以更早发现?
    ├── 仓位管理是否合理?
    ├── 止损执行是否及时?
    └── 有什么教训可以沉淀到投资体系?
```

## 七、常见风险案例库

```
A股典型风险案例参考:

财务造假类:
├── 康美药业: 300亿现金消失，存贷双高

├── 教培行业(2021双减): 市值蒸发90%+
├── 互联网(2021反垄断): 龙头跌幅50-70%
├── 光伏(2018 531政策): 板块短期暴跌
└── 教训: 政策敏感行业需降低仓位上限

商誉减值类:
├── 天神娱乐: 商誉减值导致巨亏
├── 众多影视公司: 2018年商誉减值潮
└── 教训: 商誉/净资产>30%需警惕

大股东风险:
├── 乐视网: 大股东挪用资金、关联交易
├── 暴风集团: 盲目扩张、控制人被拘
└── 教训: 关注大股东质押率、关联交易、个人风格

行业周期类:
├── 猪周期: 猪价波动导致养殖企业盈亏巨幅波动
├── 面板周期: 供需错配导致价格大幅波动
└── 教训: 周期股在盈利顶峰时PE最低，不代表便宜
```
