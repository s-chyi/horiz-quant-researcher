# 估值方法论参考手册

## 一、PE (市盈率) 估值法

### 适用范围
- 盈利稳定的成熟公司
- 不适用于亏损公司、周期性极强的公司

### 计算方法
```
PE = 股价 / EPS
   = 市值 / 净利润

PE(TTM) = 市值 / 最近4个季度净利润之和
PE(Fwd) = 市值 / 未来12个月预期净利润
PE(静态) = 市值 / 上一完整财年净利润
```

### 合理 PE 参考区间
| 公司类型 | PE 参考区间 | 说明 |
|---------|-----------|------|
| 大盘蓝筹 | 8-15x | 稳定分红，低成长 |
| 消费龙头 | 20-35x | 品牌溢价，稳定成长 |
| 高成长科技 | 30-60x | 高增速支撑 |
| 周期股 | 不适用 | PE 在底部时反而最高 |
| 银行 | 4-8x | 杠杆经营，PB更合适 |

### PE 估值陷阱
1. **周期陷阱**: 周期股在盈利顶峰时PE最低，不代表便宜
2. **一次性利润**: 非经常性损益导致PE失真
3. **利润操纵**: 通过会计手段调节利润
4. **高增长终结**: PE看起来合理，但增速正在下滑

## 二、PB (市净率) 估值法

### 适用范围
- 银行、保险等金融股
- 重资产行业 (地产、公用事业)
- 周期性行业
- 接近破产清算的公司

### 计算方法
```
PB = 股价 / 每股净资产
   = 市值 / 净资产

PB 与 ROE 的关系:
理论上 PB = (ROE - g) / (r - g)
其中 r = 股权成本, g = 永续增长率
→ ROE 越高, 合理 PB 越高
```

### 合理 PB 参考
| ROE 水平 | 合理 PB 参考 |
|---------|------------|
| 5-8% | 0.5-1.0x |
| 8-12% | 0.8-1.5x |
| 12-18% | 1.2-2.5x |
| 18-25% | 2.0-4.0x |
| >25% | 3.0-8.0x |

### PB 估值陷阱
1. **资产质量**: 净资产不代表真实价值（如过时产能、虚高商誉）
2. **隐性负债**: 表外负债导致净资产虚高
3. **轻资产公司**: 品牌、技术等无形资产不在账面上

## 三、PS (市销率) 估值法

### 适用范围
- 亏损但有收入的公司
- SaaS/互联网公司
- 高成长初期公司

### 计算方法
```
PS = 市值 / 营收
EV/Sales = (市值 + 净债务) / 营收
```

### 合理 PS 参考
| 营收增速 | 合理 PS |
|---------|--------|
| <10% | 0.5-2x |
| 10-30% | 2-5x |
| 30-50% | 5-10x |
| >50% | 10-20x |

## 四、PEG 估值法

### 适用范围
- 成长型公司
- 用于衡量"成长的性价比"

### 计算方法
```
PEG = PE / 净利润增速(%)

PEG < 0.5 → 显著低估 (需确认增速可持续)
PEG 0.5-1.0 → 合理偏低
PEG 1.0-1.5 → 合理
PEG 1.5-2.0 → 合理偏高
PEG > 2.0 → 偏贵 (除非有强催化)
```

### PEG 注意事项
1. 增速的"G"应该用未来1-2年的预期增速，而非过去增速
2. 增速的可持续性至关重要
3. 低PEG可能是因为市场预期增速下滑

## 五、EV/EBITDA 估值法

### 适用范围
- 资本密集型行业
- 跨国比较 (消除税率和资本结构差异)
- 并购估值

### 计算方法
```
EV = 市值 + 总债务 - 现金
EBITDA = 营业利润 + 折旧 + 摊销

EV/EBITDA = 企业价值 / EBITDA
```

### 合理区间参考
| 行业 | EV/EBITDA 参考 |
|-----|---------------|
| 公用事业 | 6-10x |
| 工业制造 | 8-12x |
| 消费品 | 10-16x |
| 科技 | 12-25x |
| 互联网 | 15-40x |

## 六、DCF (现金流折现) 估值法

### 核心逻辑
公司价值 = 未来所有自由现金流的折现值之和

### 关键参数

#### WACC (加权平均资本成本)
```
WACC = E/(E+D) × Ke + D/(E+D) × Kd × (1-T)

Ke = Rf + β × ERP
其中:
  Rf = 无风险利率 (10年期国债收益率, 当前约 2.0-2.5%)
  β = 个股Beta系数
  ERP = 股权风险溢价 (A股一般取 5-7%)

A股 WACC 一般范围: 8-12%
```

#### 永续增长率 (g)
```
合理范围: 1-3%
不应超过长期GDP增速
常用: 2-2.5%
```

#### FCFF (企业自由现金流)
```
FCFF = EBIT × (1-T) + 折旧&摊销 - 资本支出 - 营运资本变动
```

### DCF 敏感性分析标准格式
```
永续增长率 g
      │  WACC
      │  8%     9%     10%    11%    12%
──────┼──────────────────────────────────
 1.0% │ [值]  [值]  [值]  [值]  [值]
 1.5% │ [值]  [值]  [值]  [值]  [值]
 2.0% │ [值]  [值]  [核心] [值]  [值]
 2.5% │ [值]  [值]  [值]  [值]  [值]
 3.0% │ [值]  [值]  [值]  [值]  [值]
```

## 七、DDM (股利折现模型)

### 适用范围
- 稳定分红的成熟公司
- 银行、公用事业、消费龙头

### Gordon 增长模型
```
P = D1 / (r - g)
其中:
  D1 = 下一年预期股利
  r = 股权成本
  g = 股利永续增长率
```

## 八、重置成本法

### 适用范围
- 重资产行业 (化工、钢铁、航运)
- 周期底部估值

### 逻辑
```
公司价值 = 重建相同产能所需的投资额
当 市值 < 重置成本 时，意味着"买公司比建工厂便宜"
```

## 九、估值方法选择决策树

```
公司盈利吗？
├── 是 → ROE稳定吗？
│   ├── 是 → PE + PB + DCF
│   └── 否 → 周期性强吗？
│       ├── 是 → PB + EV/EBITDA + 重置成本
│       └── 否 → PE + PEG + DCF
└── 否 → 有收入吗？
    ├── 是 → PS + EV/Sales
    └── 否 → 市值/用户数, VC估值法
```
