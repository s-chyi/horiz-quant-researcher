# 催化剂扫描

## 适用场景

识别和评估可能改变投资标的价格或行业走势的催化事件。

## 执行步骤

### 步骤 1: 催化剂分类扫描

```
催化剂全景扫描:

├── 📊 公司层面
│   ├── 业绩相关: 财报发布 / 业绩预告 / 盈利修正
│   ├── 经营事件: 新产品发布 / 大客户获取 / 产能投放
│   ├── 资本运作: 增发 / 回购 / 分红 / 并购重组
│   ├── 管理层变动: 高管更替 / 股权激励
│   └── 股东行为: 大股东增减持 / 股权质押
│
├── 📈 行业层面
│   ├── 供需变化: 产能投放 / 需求拐点 / 库存周期
│   ├── 价格变化: 产品涨价 / 原材料价格
│   ├── 技术变革: 新技术突破 / 技术路线切换
│   └── 竞争格局: 龙头策略变化 / 新进入者
│
├── 🏛️ 政策层面
│   ├── 产业政策: 扶持 / 限制 / 准入
│   ├── 财税政策: 税收优惠 / 补贴 / 退坡
│   ├── 监管政策: 行业监管 / 反垄断 / 环保
│   └── 贸易政策: 关税 / 制裁 / 出口管制
│
└── 🌍 宏观层面
    ├── 货币政策: 降息 / 降准 / 流动性
    ├── 经济数据: GDP / PMI / CPI / 社融
    ├── 地缘政治: 国际关系 / 区域冲突
    └── 汇率变动: 人民币升值/贬值
```

### 步骤 2: 催化剂评估矩阵

```
催化剂评估表:

序号 | 催化剂描述      | 类型  | 方向 | 概率  | 影响  | 时间窗口  | 关注度
────┼────────────────┼──────┼─────┼──────┼──────┼─────────┼──────
  1  | [具体事件]      | 公司  | 正/负| [X]% | 大中小| [时间]   | ★★★★★
  2  | [具体事件]      | 行业  | 正/负| [X]% | 大中小| [时间]   | ★★★★☆
  3  | [具体事件]      | 政策  | 正/负| [X]% | 大中小| [时间]   | ★★★☆☆
  ...
```

### 步骤 3: 影响路径推演

对于高关注度催化剂，推演完整影响路径:

```
催化剂: [事件描述]

如果触发:
  一阶效应: [直接影响] (时滞: [X]天/周/月)
    → 二阶效应: [传导影响] (时滞: [X])
      → 三阶效应: [最终对标的影响] (时滞: [X])
        → 预计股价影响: [+/-X%] (基于历史类比)

如果未触发:
  → 替代情景: [...]
  → 对投资论点的影响: [是否改变核心逻辑]

历史类比:
  [类似催化剂] 在 [时间] 触发后:
    → 股价反应: [X]% / [Y]个交易日
    → 持续性: [短暂冲击 / 趋势性改变]
```

### 步骤 4: 催化剂日历

```
近期催化剂时间表:

[月份]
├── [日期] [事件] → 影响: [正/负/不确定] ★★★
├── [日期] [事件] → 影响: [正/负/不确定] ★★★★
├── [日期] [事件] → 影响: [正/负/不确定] ★★
...

重点关注:
  近1周: [最重要的催化剂]
  近1月: [最重要的催化剂]
  近1季: [最重要的催化剂]
```

### 步骤 5: ASCII 催化剂地图

```
示例:

  催化剂时间线 & 影响矩阵

  影响
  大 │     ●财报(4/25)          ●新品发布(6月)
    │          ○降息预期
  中 │  ●行业数据              ●产能投放
    │              ○关税谈判
  小 │      ○高管增持     ○分红
    │
    └────┬────┬────┬────┬────┬────→ 时间
       1周  2周  1月  2月  3月  6月

  ● 高概率(>60%)  ○ 中概率(30-60%)  △ 低概率(<30%)
  正面催化剂: 绿色  负面催化剂: 红色  不确定: 黄色
```
