# 行业链分析

## 适用场景

用户想了解某个行业/产业链的全景图谱，包括上中下游关系、利润分配、景气度传导和投资机会。

## 核心理念

**不只是画产业链图，而是找到产业链中"利润的引力中心"和"变化的传导路径"。**

## 执行步骤

### 步骤 1: 产业链全景图谱

```
[行业名称] 产业链全景

上游 (原材料/设备)        中游 (制造/加工)         下游 (应用/终端)
┌──────────────┐    ┌──────────────┐    ┌──────────────┐
│ 关键原材料    │ →  │ 核心制造环节  │ →  │ 终端应用场景  │
│ ├── 材料A    │    │ ├── 环节A    │    │ ├── 场景A    │
│ ├── 材料B    │    │ ├── 环节B    │    │ ├── 场景B    │
│ └── 材料C    │    │ └── 环节C    │    │ └── 场景C    │
│              │    │              │    │              │
│ 关键设备      │    │ 辅材/耗材    │    │ 渠道/服务    │
│ ├── 设备A    │    │ ├── 辅材A    │    │ ├── 渠道A    │
│ └── 设备B    │    │ └── 辅材B    │    │ └── 渠道B    │
└──────────────┘    └──────────────┘    └──────────────┘
```

**必须回答**:
1. 产业链共有几个核心环节？
2. 每个环节的主要玩家是谁？（上市公司标注代码）
3. 哪个环节是"卡脖子"环节？
4. 技术路线是否存在分叉？


### 步骤 3: 景气度传导链

```
景气度传导模型:

下游需求变化
  → 中游订单/排产变化 (滞后 [X] 个月)
    → 中游产能利用率变化
      → 中游扩产/缩产决策
        → 上游设备/材料需求变化 (滞后 [X] 个月)
          → 上游价格变化
            → 反馈至中游成本 (周期性循环)

当前位置: 产业链景气度处于 [上行/顶部/下行/底部] 阶段
领先指标: [哪些指标可以提前判断景气拐点]
```
