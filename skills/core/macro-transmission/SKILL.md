# 宏观传导链分析

## 适用场景

用户关注宏观经济数据、货币政策、财政政策等宏观变量，想了解对投资标的的影响传导路径。

## 核心原则

**宏观分析的关键不是解读数据本身，而是推演传导链：宏观变量 → 行业 → 公司 → 股价。**

## 执行步骤

### 步骤 1: 宏观环境画像

```
当前宏观环境:

经济周期位置: [复苏/过热/滞胀/衰退]
├── GDP 增速: [X]% (趋势: ↑/→/↓)
├── PMI: [X] (荣枯线 50 以上/下)
├── CPI: [X]% (通胀/通缩)
├── PPI: [X]% (上游价格)
└── 社融存量增速: [X]% (信用扩张/收缩)

货币政策:
├── 政策定调: [宽松/中性/收紧]
├── LPR 1年: [X]%, 5年: [X]%
├── 存款准备金率: [X]%
├── 最近操作: [降息/降准/MLF/OMO]
└── 市场流动性: [充裕/适度/偏紧]

财政政策:
├── 赤字率: [X]%
├── 专项债额度: [X] 万亿
├── 重点投向: [基建/科技/民生]
└── 减税降费: [有/无], 规模 [X]

汇率:
├── 美元/人民币: [X]
├── 趋势: [升值/稳定/贬值]
└── 央行态度: [...]
```

### 步骤 2: 宏观传导链推演

```
传导链模型:

[宏观变量变化]
  │
  ├── 一阶传导 (直接影响)
  │   ├── 利率下降 → 融资成本降低 → 企业利润改善
  │   ├── 汇率贬值 → 出口竞争力增强 → 出口订单增加
  │   └── 财政发力 → 基建投资增加 → 建材/工程机械需求
  │
  ├── 二阶传导 (间接影响)
  │   ├── 融资成本降低 → 房贷利率下降 → 购房需求释放 → 家电/建材
  │   ├── 出口增加 → 制造业开工率提升 → 上游原材料需求
  │   └── 基建投资 → 就业改善 → 消费回暖
  │
  └── 三阶传导 (长期效应)
      ├── 经济复苏预期 → 风险偏好提升 → 股市估值修复
      ├── 产能扩张 → 未来可能过剩 → 下一轮周期种子
      └── 通胀压力积累 → 政策转向预期 → 利率预期变化

各阶传导时滞:
  一阶: [X] 周/月 (最快见效)
  二阶: [X] 月 (中期显现)
  三阶: [X] 季度/年 (长期效应)
```

