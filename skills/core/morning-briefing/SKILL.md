# 晨报/复盘

## 适用场景

盘前: 快速了解隔夜要闻、海外市场、资金动向，为当日交易做准备。
盘后: 复盘当日行情，分析资金行为，展望次日。

## 执行步骤

### 步骤 1: 隔夜/全球市场速览

```
🌍 全球市场速览 ([日期])

美股:
  道琼斯: [点位] ([+/-X]%)  纳斯达克: [点位] ([+/-X]%)
  标普500: [点位] ([+/-X]%)  VIX: [X]
  要点: [1-2句关键信息]

欧股:
  英国FTSE: [+/-X]%  德国DAX: [+/-X]%  法国CAC: [+/-X]%

亚太(昨收):
  日经225: [+/-X]%  韩国KOSPI: [+/-X]%  恒生: [+/-X]%

商品:
  原油(WTI): $[X] ([+/-X]%)  黄金: $[X] ([+/-X]%)
  铜: $[X] ([+/-X]%)

外汇:
  美元指数: [X]  USD/CNY: [X]  USD/CNH: [X]

加密:
  BTC: $[X] ([+/-X]%)  ETH: $[X] ([+/-X]%)
```

  成交额: [X]亿 (vs 前日 [X]亿, [放量/缩量])
  涨停/跌停: [X]/[X]  涨跌比: [X]

  北向资金: 净流入/出 [X]亿 (近5日累计: [X]亿)

  主力资金: 净流入/出 [X]亿
  ETF资金: [大盘ETF/行业ETF 申赎情况]

板块热点:
  领涨: [板块1] (+[X]%) / [板块2] (+[X]%) / [板块3] (+[X]%)
  领跌: [板块1] (-[X]%) / [板块2] (-[X]%) / [板块3] (-[X]%)

## 数据源

1. 新浪财经 API → 海外市场、A 股行情、资金流向
2. 腾讯行情 API → 实时/延迟行情
3. Web 搜索 → 最新新闻和公告
4. horiz-datasource-api → 最新路演纪要和研报观点
