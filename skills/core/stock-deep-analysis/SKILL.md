# 个股深度分析

## 适用场景

用户想要深入了解某只股票/公司的全面分析。这是最常用的技能，必须展示超越竞品的推理深度。

## 执行步骤

### 步骤 1: 公司定位与背景

**必须回答**:
- 公司做什么业务？（用一句话说清核心商业模式）
- 在行业中处于什么位置？（龙头/挑战者/边缘玩家）
- 当前市值/PE/PB 的大致水平
- 近期是否有重大事件？

**数据源优先级**:
1. horiz-datasource-api: 查询相关路演纪要和研报
2. 腾讯行情 API: 获取实时行情
3. Web 搜索: 补充最新新闻和公告

### 步骤 2: 五维分析框架

#### 维度一: 基本面分析 (权重 35%)

```
业绩拆解链:
  营收 → 按业务线拆解 → 各线的量×价
    → 增长驱动力识别（哪条线贡献最大增量？）
    → 可持续性判断（一次性 vs 持续性）

利润分析链:
  毛利率 → 成本结构（原材料/人工/折旧）
    → 费用率趋势（销售/管理/研发/财务）
    → 归母净利润 → ROE DuPont 拆解

资产负债分析:
  现金及等价物 → 有息负债 → 净负债率
    → 经营性现金流 vs 净利润（含金量）
    → 资本开支趋势（扩张期 vs 收获期）
```

**DuPont 拆解必须展示**:
```
ROE = 净利率 × 资产周转率 × 权益乘数
     = [X]% × [Y] × [Z]
变动归因: 主要由 [哪个因子] 驱动，原因是 [...]
```

#### 维度二: 行业与竞争 (权重 25%)

```
行业分析链:
  行业规模与增速 → 生命周期阶段（导入/成长/成熟/衰退）
    → 竞争格局（CR3/CR5、HHI指数方向）
    → 行业壁垒（技术/规模/品牌/政策）

公司竞争优势:
  波特五力分析:
    - 现有竞争者威胁: [高/中/低] ← [原因]
    - 新进入者威胁: [高/中/低] ← [原因]
    - 替代品威胁: [高/中/低] ← [原因]
    - 供应商议价能力: [高/中/低] ← [原因]
    - 买方议价能力: [高/中/低] ← [原因]

护城河评估:
  ├── 成本优势（规模/技术/资源）
  ├── 转换成本（客户粘性）
  ├── 网络效应（平台型）
  ├── 无形资产（品牌/专利/牌照）
  └── 评级: [无/窄/宽] ← [依据]
```

#### 维度三: 技术面分析 (权重 15%)

```
趋势判断:
  均线系统（MA5/10/20/60/120/250）→ 多空排列状态
    → 当前处于趋势的哪个阶段
    → 关键支撑位和压力位

量价关系:
  成交量趋势 → 量价配合度 → 放量/缩量的含义
    → 换手率异常检测

技术形态:
  当前形态识别 → 历史相似形态的后续走势统计
```

#### 维度四: 资金面分析 (权重 15%)

```
主力资金:
  近5/10/20日主力净流入 → 流入强度趋势
    → 与股价走势的背离/共振

北向资金:
  持股数量变化 → 占流通股比例 → 近期增减趋势
    → 北向资金偏好信号

融资融券:
  融资余额趋势 → 融资买入占比 → 杠杆情绪指标

机构持仓:
  基金重仓变化（季报） → 机构共识度变化
```

#### 维度五: 估值分析 (权重 10%)

```
绝对估值:
  DCF 简易估算 → 关键假设（增速/WACC/永续增长率）
  → 敏感性分析

相对估值:
  PE/PB/PS → 与历史分位数对比 → 与同行对比
    → 估值溢价/折价的合理性

PEG 估值:
  PEG = PE / 盈利增速 → >1 偏贵 / <1 偏便宜
    → 但需考虑增速的可持续性
```

### 步骤 3: 矛盾识别与辩证

**强制执行**: 在五维分析中找出至少一对矛盾:

```
常见矛盾模式:
├── 基本面向好 but 技术面走弱 → 可能是市场预期已充分
├── 资金持续流入 but 估值偏高 → 可能是惯性 or 有催化剂
├── 行业景气向上 but 公司业绩不达预期 → 可能是竞争加剧
├── 财务数据漂亮 but 现金流恶化 → 可能是应收账款风险
└── ...

处理方式: 不回避矛盾，显式展示并给出权衡判断
```
