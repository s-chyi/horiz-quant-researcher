# 估值框架

## 适用场景

用户想评估某只股票/公司是否被高估或低估。本技能提供多方法交叉验证，避免单一方法的偏差。

## 核心原则

**估值不是精确计算，而是框定"合理区间"。任何给出精确目标价的分析都是伪精确。**

## 执行步骤

### 步骤 1: 估值方法选择

根据公司特征选择适用的估值方法:

```
方法选择矩阵:

公司类型          适用方法                    不适用方法
─────────────   ────────────────────────   ─────────────
成熟盈利型       PE / PB / DCF / DDM        PS / EV/Sales
高速成长型       PEG / PS / EV/EBITDA       PE(失真) / DDM
周期型           PB / EV/EBITDA / 重置成本   PE(波动大)
互联网/SaaS     PS / EV/ARR / 用户价值      PE / PB
金融股           PB / P/EV / DDM            PE(杠杆扭曲)
重资产/公用事业  DCF / EV/EBITDA / DDM      PS
亏损公司         PS / EV/Sales / 市值/GMV   PE / PB(失真)
```

### 步骤 2: 绝对估值 — DCF 模型

```
DCF 估值模型:

假设:
├── 预测期: [N] 年
├── 营收增速:
│   Year 1: [X]% → Year 2: [X]% → ... → Year N: [X]%
├── 净利率假设: [X]% → 逐步变化至 [X]%
├── 资本开支/折旧比: [X]
├── WACC:
│   ├── 无风险利率: [X]% (10年国债收益率)
│   ├── 股权风险溢价: [X]%
│   ├── Beta: [X]
│   ├── 股权成本 = [X]%
│   ├── 债务成本 = [X]%
│   ├── 资本结构: 股权 [X]% / 债务 [X]%
│   └── WACC = [X]%
├── 永续增长率: [X]%
└── 注: 以上每个假设都需要说明依据

现金流预测表:
Year     | 营收    | EBIT   | FCFF   | 折现FCFF
─────────┼────────┼────────┼────────┼────────
Year 1   |        |        |        |
Year 2   |        |        |        |
...      |        |        |        |
Year N   |        |        |        |
终值      |        |        |        |
─────────┼────────┼────────┼────────┼────────
合计      |                          | [X]亿

股权价值 = 企业价值 - 净债务 = [X]亿
每股价值 = [X]元

敏感性分析:
        WACC
        [X-2]%  [X-1]%  [X]%   [X+1]%  [X+2]%
g=X-1%  [价格]  [价格]  [价格] [价格]   [价格]
g=X%    [价格]  [价格]  [核心] [价格]   [价格]
g=X+1%  [价格]  [价格]  [价格] [价格]   [价格]
```

### 步骤 3: 相对估值 — 可比公司

```
可比公司估值:

可比公司选择标准:
├── 同行业/相似业务模式
├── 相似市值量级
├── 相似增长阶段
└── 相似市场定位

可比公司估值表:
公司      | 市值   | PE(TTM)| PE(Fwd)| PB    | PS    | EV/EBITDA
─────────┼───────┼───────┼───────┼──────┼──────┼─────────
可比A     |       |       |       |      |      |
可比B     |       |       |       |      |      |
可比C     |       |       |       |      |      |
可比D     |       |       |       |      |      |
─────────┼───────┼───────┼───────┼──────┼──────┼─────────
中位数    |       |       |       |      |      |
目标公司  |       |       |       |      |      |
溢价/折价 |       |       |       |      |      |

溢价/折价合理性分析:
  如果目标公司估值高于中位数 → 溢价是否由 [增速/壁垒/品质] 支撑？
  如果目标公司估值低于中位数 → 折价是否反映了 [风险/瑕疵] 还是被低估？
```

### 步骤 4: 历史估值分位

```
历史估值分位:

指标    | 当前值  | 近3年分位 | 近5年分位 | 近10年分位
───────┼───────┼─────────┼─────────┼──────────
PE(TTM) |       | [X]%    | [X]%    | [X]%
PE(Fwd) |       | [X]%    | [X]%    | [X]%
PB      |       | [X]%    | [X]%    | [X]%
PS      |       | [X]%    | [X]%    | [X]%

历史分位解读:
  > 80% → ⚠️ 偏贵区域，需要强催化剂支撑
  50-80% → 中性
  20-50% → 偏便宜，但需确认不是 "价值陷阱"
  < 20% → 低估区域 OR 基本面恶化的信号

注意: 分位数不等于买卖信号。低分位可能是因为基本面恶化（Value Trap），
     高分位可能是因为公司进入了新的增长阶段（Re-rating）。
```
