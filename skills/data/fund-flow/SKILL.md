# 资金流向分析

## 适用场景

跟踪和分析各类资金流向数据，判断资金行为对股价/板块的影响。

## 核心原则

**资金流向是「结果」不是「原因」。不要因为北向资金买入就看多，要理解资金背后的逻辑。同时注意资金流向数据有滞后性和噪音。**

```
⚠️ 重要提醒:
资金流向数据有以下局限:
├── 主力资金统计基于成交大小单推算，并非真实持仓变动
├── 北向资金T+0数据可能有尾盘异动干扰
├── 融资融券数据有1日延迟
└── 不应单独依赖资金流向做投资决策
```

## ⚠️ 数据局限性说明

```
当前可获取:
├── 🟢 北向资金日度数据 (Web搜索/新浪)
├── 🟢 融资融券日度数据 (Web搜索/东方财富)
├── 🟡 主力资金推算 (东方财富，准确性有限)
└── 🟡 板块资金流向 (准确性有限)

当前缺失 (需工程团队集成):
├── 🔴 北向资金实时盘中数据
├── 🔴 ETF实时申赎数据
├── 🔴 个股北向持仓明细 (逐日)
├── 🔴 基金重仓股数据 (季度更新)
└── 🔴 大宗交易和龙虎榜数据
```

## 数据源

1. Web 搜索 → 北向资金、融资融券、龙虎榜 (东方财富/同花顺)
2. 新浪财经 → 板块资金流向
3. horiz-datasource-api → 路演纪要中的资金相关讨论
