# 市场行情数据

## 适用场景

获取实时或历史市场行情数据，支撑其他分析技能的数据需求。

## 核心原则

**行情数据是分析的基础设施。确保数据的及时性、准确性，并明确标注数据的延迟和来源。**

## 当前可用数据源

### 新浪财经 API (免费，无需认证)

```
可用接口:

1. 实时行情 (A股):
   URL: https://hq.sinajs.cn/list=sh600519,sz000858
   返回: 实时价格/涨跌幅/成交量/买卖盘口
   延迟: ~3秒 (盘中)
   数据质量: 🟢

2. 实时行情 (港股):
   URL: https://hq.sinajs.cn/list=hk00700
   数据质量: 🟢

3. 实时行情 (美股):
   URL: https://hq.sinajs.cn/list=gb_aapl
   数据质量: 🟢 (交易时段) / 🟡 (盘前盘后)

4. 指数:
   URL: https://hq.sinajs.cn/list=s_sh000001,s_sz399001,s_sz399006
   覆盖: 上证/深成指/创业板/科创50等
   数据质量: 🟢
```

### 腾讯行情 API (免费，无需认证)

```
可用接口:

1. 实时行情:
   URL: https://qt.gtimg.cn/q=sh600519
   数据质量: 🟢

2. 分时数据:
   URL: https://web.ifzq.gtimg.cn/appstock/app/minute/query?code=sh600519

   URL: https://web.ifzq.gtimg.cn/appstock/app/fqkline/get?param=sh600519,day,,,320,qfq
```

```
   ├── 优先使用: 新浪实时行情 API
```
