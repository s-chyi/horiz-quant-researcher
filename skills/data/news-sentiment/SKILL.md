# 新闻舆情分析

## 适用场景

实时获取和分析与投资标的/行业相关的新闻和市场舆情，评估市场情绪对股价的潜在影响。

## 核心原则

**新闻本身不重要，重要的是：(1) 市场是否已经Price in，(2) 新闻改变了什么预期，(3) 后续可能的发展路径。避免被噪音带偏。**

## 执行步骤

### 步骤 1: 新闻收集

```
新闻收集策略:

搜索关键词组合:
├── 一级关键词: [公司名/代码/行业名]
├── 二级关键词: [政策/业绩/并购/产品/管理层...]
└── 排除词: [广告/招聘/...]

数据源优先级:
1. horiz-datasource-api → 微信文章、行业研报 (私域数据)
2. Tavily Web搜索 → 最新新闻、政策、公告
3. 新浪财经 → 个股新闻、公告
4. 东方财富 → 互动平台、研报速递

搜索时间范围:
├── 默认: 近7天
├── 紧急分析: 近24小时
└── 全面分析: 近30天
```

### 步骤 2: 新闻分类与影响评估

```
新闻汇总 ([标的/行业]):

🔴 重大负面 (可能引发股价下跌 >3%):
1. [日期] [来源] [新闻标题]
   ├── 摘要: [1-2句]
   ├── 影响评估: [具体影响路径]
   ├── 影响时间: [短期冲击/中期影响/长期改变]
   └── 是否已Price in: [是/部分/否] — 依据: [...]

🟡 值得关注 (可能影响股价 1-3%):
1. [日期] [来源] [新闻标题]
   ├── 摘要: [1-2句]
   ├── 影响评估: [正面/负面/中性]
   └── 后续观察点: [...]

🟢 重大正面 (可能推动股价上涨 >3%):
1. [日期] [来源] [新闻标题]
   ├── 摘要: [1-2句]
   ├── 影响评估: [具体受益路径]
   └── 持续性评估: [一次性/持续性催化]

⚪ 噪音过滤 (市场关注但实质影响有限):
1. [新闻标题] ← 为什么是噪音: [...]
2. [新闻标题] ← 为什么是噪音: [...]
```

### 步骤 3: 市场情绪分析

```
市场情绪评估:

├── 📊 量化指标:
│   ├── 新闻情绪得分: [X]/100 (0=极度恐慌, 100=极度贪婪)
│   │   └── 基于: 正面新闻[X]条 vs 负面新闻[X]条
│   ├── 社交媒体热度: [高/中/低]
│   │   └── 基于: 雪球/东财股吧讨论量
│   ├── 分析师情绪: [乐观/中性/悲观]
│   │   └── 基于: 近期评级调整 [上调X/维持X/下调X]
│   └── 搜索热度: [上升/稳定/下降]
│
├── 📈 情绪与价格关系:
│   ├── 当前情绪位置: [过度乐观/乐观/中性/悲观/过度悲观]
│   ├── 情绪-价格背离: [有/无]
│   │   └── [如有]: [情绪X但股价Y，通常意味着Z]
│   └── 历史经验: [类似情绪水平时，后续走势通常是X]
│
├── 🔍 关键叙事追踪:
│   ├── 当前市场主流叙事: [描述市场讲的"故事"]
│   ├── 叙事强度: [增强/稳定/减弱]
│   ├── 叙事风险: [什么可能打破当前叙事]
│   └── 替代叙事: [市场可能转向的新叙事]
│
└── ⚠️ 情绪极端信号:
    ├── 极度乐观信号: [列出，如有]
    │   → 反向指标: 可能意味着短期见顶
    ├── 极度悲观信号: [列出，如有]
    │   → 反向指标: 可能意味着短期见底
    └── 当前是否触发极端信号: [是/否]
```

### 步骤 4: 政策/监管动态专题

```
政策/监管动态:

├── 近期政策变化:
│   ├── [日期] [政策名称]
│   │   ├── 核心内容: [...]
│   │   ├── 影响行业: [...]
│   │   ├── 影响方向: [利好/利空/中性]
│   │   └── 影响时滞: [即时/1-3月/3-6月/6-12月]
│   └── ...
│
├── 政策展望:
│   ├── 近期可能出台的政策: [...]
│   ├── 重要会议/文件: [日期] [事件]
│   └── 政策方向判断: [...]
│
└── 监管风险:
    ├── 行业监管趋势: [收紧/稳定/放松]
    ├── 特定监管风险: [如有]
    └── 合规成本变化: [上升/稳定/下降]
```

### 步骤 5: 舆情时间线

```
舆情时间线 (近[X]天):

[日期1]
  ├── 📰 [新闻/公告标题] → [正面/负面] ★★★
  ├── 📊 股价反应: [+/-X]%
  └── 分析: [市场如何解读]

[日期2]
  ├── 📰 [新闻标题] → [正面/负面] ★★
  ├── 📊 股价反应: [+/-X]%
  └── 分析: [...]

[日期3]
  ├── 🏛️ [政策发布] → [影响方向] ★★★★
  ├── 📊 板块反应: [+/-X]%
  └── 分析: [...]

舆情趋势: [近期舆情整体向好/恶化/平稳]
关键转折点: [如有，什么事件可能改变舆情走向]
```

```
示例:

  市场情绪仪表盘

  整体情绪: ██████████████████░░ 72/100 (偏乐观)

  分项指标:
  新闻面   ████████████████░░░░ 80/100 正面主导
  分析师   ██████████████░░░░░░ 70/100 偏乐观
  社交媒体 ██████████████████░░ 75/100 活跃讨论
  资金面   ████████████░░░░░░░░ 60/100 中性偏积极

  情绪趋势 (近5日):
  72 │           ●
  68 │        ●
  65 │     ●
  60 │  ●
  55 │●
     └──┬──┬──┬──┬──→
       D-4 D-3 D-2 D-1 今日

  → 情绪持续升温，注意过热风险

  ⚠️ 提醒: 情绪指标为辅助参考，不构成买卖建议。
  极端情绪往往是反向指标。
```

## 数据源

1. **horiz-datasource-api** → 微信文章、研报观点 (私域数据)
2. **Tavily Web搜索** → 最新新闻、政策公告
3. **新浪财经** → 个股新闻、公告
4. Web搜索 → 雪球/东财股吧讨论、分析师观点
