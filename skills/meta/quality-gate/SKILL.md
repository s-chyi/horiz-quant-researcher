# 质量门控

## 功能

所有分析技能的最终输出前必须通过本门控检查。门控不通过则回退补充，不得直接输出。

## 检查清单

### A. 推理链完整性 (权重 30%)

- [ ] **框架声明**: 是否在开头明确了使用的分析框架？
- [ ] **步骤可见**: 推理过程是否分步展示？（最少 3 步）
- [ ] **逻辑连贯**: 每步结论是否基于前步推理？（无跳跃）
- [ ] **假设显式**: 关键假设是否被明确列出？

**不通过示例**: "宁德时代是好公司，值得买入" ← 无推理过程
**通过示例**: "从 DuPont 拆解看，ROE 22% 由净利率 10%×周转率 0.85×杠杆 2.6 驱动，其中净利率同比下降 2pp 主要由于碳酸锂价格回落传导滞后..."

### B. 多维度覆盖 (权重 25%)

- [ ] **维度数量**: 是否覆盖 ≥3 个分析维度？
- [ ] **交叉验证**: 不同维度的结论是否进行了交叉检验？
- [ ] **矛盾处理**: 维度间矛盾是否被显式讨论？

**维度清单** (根据问题类型，至少覆盖 3 个):
1. 基本面（财务/业务/管理层）
2. 技术面（趋势/形态/指标）
3. 资金面（主力流向/北向/融资）
4. 估值面（绝对/相对/历史分位）
5. 情绪面（市场共识/预期差）
6. 政策面（监管/产业政策/贸易）
7. 宏观面（经济周期/利率/汇率）
8. 行业面（竞争格局/产业趋势/壁垒）

### C. 数据溯源 (权重 20%)

- [ ] **关键数据标注**: 每条关键数字是否标注了来源？
- [ ] **时效性**: 数据是否注明了截止日期？
- [ ] **质量分级**: 是否按🟢🟡🔴标注了数据可靠度？
- [ ] **无编造**: 没有使用未经验证的数据？

**不通过示例**: "公司营收增长 30%" ← 无来源
**通过示例**: "公司 2025Q3 营收 421 亿元，YoY +28.3% —— 来源: 公司季报, 2025-10-28 披露"

### D. 反面论证 (权重 15%)

- [ ] **Devil's Advocate**: 是否包含至少一个反面论点？
- [ ] **风险识别**: 是否列出了主要风险因子？
- [ ] **情景分析**: 是否给出了悲观情景？

**不通过示例**: 全文只有正面论述
**通过示例**: "然而需要注意: 1) 海外扩产可能遭遇贸易壁垒; 2) 钠电池技术替代风险..."

### E. 合规标注 (权重 10%)

- [ ] **免责声明**: 末尾是否包含"不构成投资建议"标注？
- [ ] **无越界**: 是否避免了直接"买入/卖出"建议？
- [ ] **模型标注**: 是否标注了 AI 辅助生成？

## 评分与处理

| 总分 | 处理 |
|------|------|
| ≥85 分 | ✅ 通过，直接输出 |
| 70-84 分 | ⚠️ 补充后输出（标注薄弱维度） |
| <70 分 | ❌ 回退重做（不得输出） |

## 快速自检提示词

在生成分析前，用以下内部检查:

```
[内部质量检查]
1. 我的推理链有几步？→ 需要≥3步
2. 我覆盖了哪些维度？→ 需要≥3个
3. 哪些数据没有标注来源？→ 需要全部标注
4. 反面论点在哪里？→ 需要≥1个
5. 免责声明加了吗？→ 必须
```
